Документ "Перемещение денег"

Перемещения денежных средств отражаются с помощью документа "Перемещения денег", а также кассовых и банковских документов: "Расход со счета", "Поступление на счет", "Расход из кассы", "Поступление в кассу" – с соответствующими видами операций.

Документы перемещения денежных средств содержатся в журнале "Перемещения денег".

В этой справке:

  • Перемещение денег между кассами
  • Перемещение денег между кассами и счетами организации
  • Денежные переводы "в пути"

 

Перемещение денег между кассами

Перемещение денег между кассами организации отражается с помощью документа "Перемещение денег".

В документе необходимо заполнить следующие реквизиты:

  • Организация, для которой оформляется документ. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Валюта. Указывается валюта денежных средств. Выбирается из справочника "Валюты".
  • Сумма. Указывается сумма перемещения в валюте денежных средств.
  • Статья движения денежных средств, по которой осуществляется перемещение. Выбирается из справочника "Статьи движения денежных средств".
  • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)".
  • Откуда. Касса-отправитель из которой осуществляется перемещение. Становится доступным для указания при типе денежных средств - наличные. Выбирается из справочника "Кассы".
  • Куда. Касса получателя, в которую осуществляется перемещение. Становится доступным для указания при типе денежных средств - наличные. Выбирается из справочника "Кассы"".
  • Комментарий. Указывается комментарий к документу.


Перемещение денег между кассами и счетами организации

Перемещения денег между кассами и счетами отражаются с помощью следующих документов:

  • Из кассы на счет. Создается документ "Расход из кассы" с видом операции "Взнос наличными в банк". При проведении документа формируются движения: расход денежных средств из кассы, приход денежных средств на банковский счет организации.

  • Из банка в кассу. Создается документ "Поступление в кассу" с видом операции "Получение наличных в банке". При проведении документа формируются движения: расход денежных средств из кассы и приход денежных средств на банковский счет организации.

  • Между счетами организации. Используется документ "Расход со счета" с видом операции "Перевод на другой счет организации".


Денежные переводы "в пути"

В случае если необходимо учитывать временные промежутки (задержки в выполнении операций банком) при перемещении денежных средств на другой счет, взнос, снятие наличных с расчетного счета в банке, необходимо выполнить дополнительную настройку.

В разделе Настройки – Еще больше возможностей – Деньги выполняется переход по ссылке "Переводы в пути". В открывшейся форме создается новая настройка для конкретной организации.

В этом случае для корректного отражения операций перемещения денежных средств документы перемещения необходимо вводить парами:

  • Перемещение из кассы на банковский счет отражается документом "Расход из кассы" и документом "Поступление на счет".
  • Перемещение с банковского счета в кассу отражается документом "Поступление в кассу" и документом "Расход со счета".
  • Перемещение между банковскими счетами организации отражается документом "Расход со счета" и документом "Поступление на счет".

 Перемещения денежных средств отражается через 57-й счет "Переводы в пути" с учетом временных промежутков.
