Документ "Инвентаризация запасов"


Возможности и реквизиты данного документа отличаются в приложениях "1С:УНФ" и "1С:Розница". 
Если в справке напротив возможности или элемента интерфейса есть иконка, то описанная опция доступна только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Документ предназначен для отражения результатов проведения инвентаризации на складе или в производстве. Документ "Инвентаризация запасов" не формирует движений.

В этой справке:

  • Заполнение документа 
  • Ввод на основании

 

 

Заполнение документа

Заполнение документа может быть произведено автоматически, вручную или из формы подбора товаров. Автоматическое заполнение производится по команде "Заполнить по остаткам на складе". При этом в табличную часть попадают товары, которые есть на остатках на выбранной структурной единице и соответствуют условиям проведения инвентаризации (сворачиваемая группа "Номенклатура в"):

  • Номенклатура в списке. В данном поле можно выбрать один или несколько элементов номенклатуры.
  • Номенклатура в категориях. Список категорий номенклатуры, в соответствии с которыми классифицированы элементы оменклатуры.
  • Номенклатура в иерархии (группы). В данном поле можно ыбрать группы иерархии справочника, в которые входит номенклатура.

Если условие не задано, оно не используется. 

Реквизиты документа:

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ.
  • Склад. В этом поле необходимо указать структурную единицу - склад или производственное подразделение, в котором проводится инвентаризация.
  • Ячейка. В этом поле необходимо указать ячейку склада, в которой проводится инвентаризация
  • Комментарий. Произвольный комментарий к документу.

Табличная часть "Запасы" содержит реквизиты:

  • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
  • Характеристика. При необходимости в данном поле указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если при настройке параметров учета указано ведение учета по характеристикам номенклатуры. Выбирается из справочника "Характеристики номенклатуры".
  • Партия. Указывается партия номенклатурной позиции. Поле доступно только в том случае, если в настройках учета указано ведение учета по партиям номенклатуры. Выбирается из справочника "Партии".
  • Ед. Единица измерения, в которой вводится количество номенклатуры. 
  • Отклонение. Разница между фактическими данными и данными учета.
  • Кол-во. Указывается количество единиц измерения номенклатурной позиции по результатам инвентаризации.
  • Кол-во учет. Указывается количество единиц измерения номенклатурной позиции по данным учета.
  • Цена. Цена остатка запаса.
  • Сумма. Сумма остатка запаса по данным инвентаризации.
  • Сумма учет. Сумма остатка запаса по данным учета.

 

Ввод на основании

Документ "Инвентаризация запасов" не формирует движений. На основании него могут быть сформированы документы:

  • "Оприходование запасов" для оприходования излишков;
  • "Списание запасов" для списания недостач;
  • "Пересортица запасов" при обнаружении пересортицы.
  • "Отчет о розничных продажах" для отражения розничных продаж в неавтоматизированной торговой точке (НТТ);
  • "Ввод начальных остатков".

Состав табличных полей этих документов будет определяться результатами проведенной инвентаризации.

В документ "Оприходование запасов" вносится излишек номенклатурных позиций, выявленный в результате инвентаризации, в документ "Списание запасов" – обнаруженная недостача.

В документ "Пересортица запасов" вносится данные об излишках и недостачах, выявленных в результате инвентаризации. Данные сопоставляются между собой, и выявляется пересортица запасов.

В документ "Отчет о розничных продажах" вносится разница между учетным количеством товара, зафиксированным в информационной базе, и фактическим количеством, выявленным в результате инвентаризации. Это количество товара будет считаться проданным.

Также на основании документа инвентаризации могут быть введены начальные остатки по разделу "Запасы".

После проведения этих документов количество номенклатурных позиций на складе установится равным реальному количеству.
