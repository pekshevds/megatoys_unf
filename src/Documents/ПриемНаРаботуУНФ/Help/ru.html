Документ "Прием на работу"


Возможности и реквизиты данного документа отличаются в приложениях "1С:УНФ" и "1С:Розница". 
Если в справке напротив возможности или элемента интерфейса есть иконка, то описанная опция доступна только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Документ регистрирует прием сотрудников на работу, а также способ и размер оплаты труда.

Реквизиты:

  • Организация. Организация, на работу в которую принимается сотрудник. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Комментарий. Указывается любая дополнительная информация к документу.

Табличная часть "Сотрудники" содержит информацию о местах работы и условиях труда сотрудников. Реквизиты табличной части:

  • Сотрудник, принимаемый на работу. Выбирается из справочника "Сотрудники". 
  • Период - дата приема.
  • Подразделение. В данном поле указывается подразделение, в которое принимается сотрудник. Выбирается из справочника "Структурные единицы".
  • Должность. В данном поле указывается должность, на которую принимается сотрудник. Выбирается из справочника "Должности".
  • Занимаемых ставок - количество занимаемых ставок. Значение не используется при расчете заработной платы, используется только для анализа занятых ставок по штатному расписанию.
  • График работы - график работы сотрудника. Выбирается из справочника "Графики работы".

Физические лица, работающие по совместительству, могут быть приняты на работу неограниченное число раз. Для этого необходимо завести для них в справочнике "Сотрудники" несколько элементов.

Табличная часть "Начисления и удержания" содержит информацию о способах оплаты труда сотрудников. Реквизиты табличной части:

  • Вид начисления (удержания) - плановое начисление или удержание сотрудника. Выбирается из справочника "Виды начислений и удержаний".
  • Тип. Может принимать одно из двух значений: "Начисление" или "Удержание". Заполняется автоматически по данным вида начисления (удержания).
  • Сумма - размер планового начисления или удержания в валюте, указанной в реквизите "Валюта" табличной части.
  •  Валюта - в данном поле указывается валюта планового начисления или удержания. Выбирается из справочника "Валюты".
  •  Счет затрат - счет, на который будут отнесены затраты.

 

 Налоги

Табличная часть  "Налоги". Табличная часть содержит информацию о плановых налогах на доходы. Табличная часть отображается на форме, если в настройках параметров учета "Персонал" включено  "Расчет НДФЛ и взносов".

  • Вид начисления (удержания) - налог. Выбирается из справочника "Виды начислений и удержаний".
  • Валюта - в данном поле указывается валюта начисления налога. Выбирается из справочника "Валюты".
