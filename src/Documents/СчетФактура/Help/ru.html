Документ "Счет-фактура"


Возможности и реквизиты данного документа отличаются в приложениях "1С:УНФ" и "1С:Розница". 
Если в справке напротив поля или элемента интерфейса есть иконка, то элемент доступен только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Документ предназначен для выписки счета-фактуры при реализации товаров, работ и услуг или при получении аванса (предоплаты) от покупателей, а также для учета номеров грузовых таможенных деклараций.

В этой справке:

  • Оформление счета-фактуры
  • Печать

 Важно! Отображение некоторых полей документа зависит от включенных возможностей программы. Активировать дополнительные возможности можно в разделе Настройки – Еще больше возможностей.

 

 

Оформление счета-фактуры

Документ "Счет-фактура" оформляется на основании документов:

  • "Расходная накладная",  "Акт выполненных работ" или  "Заказ-наряд" при реализации товаров, работ и услуг.
  • "Поступление на счет" и "Поступление в кассу" при получении аванса (предоплаты) от покупателей.
  • "Отчет комитенту" и  "Отчет о переработке" при использовании соответствующих разделов учета.

При создании нового счета-фактуры в зависимости от документа-основания определяется вид документа, автоматически заполняется в поле "Операция". Счет-фактура может быть выписан:

  • На продажу,
  • На аванс,
  • На суммовые разницы. 

Из списка документов "Счета-фактуры" по умолчанию создается счет-фактура на продажу.

Создать счет-фактуру можно непосредственно из подвала документа-основания (например, расходной накладной) по гиперссылке "Создать".

В счет-фактуру переносятся сведения из соответствующего документа-основания. Номер и дата присваиваются документу автоматически при записи.

Состав реквизитов счета-фактуры:

  • Организация, от которой оформляется счет-фактура. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Вид операции. Указывается вид операции документа. Предлагается выбор из списка: продажа, аванс или суммовые разницы.
  • Основание. Указывается документ-основание выдачи счета-фактуры. В зависимости от выбранного вида операции документом-основанием может быть:.
  • Контрагент. Указывается покупатель, которому формируется счет-фактура. Выбирается из справочника "Контрагенты".
  • Договор покупателя, по которому формируется счет-фактура. Выбирается из справочника "Договоры контрагентов".
  • Комментарий. Указывается любая дополнительная информация к документу.
  • Сумма, НДС, Всего. Поля автоматически заполняются по данным документа-основания: стоимость, сумма налога на добавленную стоимость, итоговая стоимость. Группа реквизитов располагается отдельно на сером фоне.

По гиперссылке "Документ оплаты" вызывается форма выбора платежно-расчетного документа.


Печать счетов-фактур

Для документа "Счет-фактура" предусмотрены печатные формы:

  • счет-фактура;

  • универсальный передаточный документ (УПД);

  • универсальный передаточный документ (УПД) с факсимиле;

Если необходимо указать дополнительные сведения для печатной формы (например, подписи ответственных лиц) разверните группу "Дополнительно" и перейдите по ссылке "Реквизиты печати".

Если оставить поля реквизитов печати пустыми, сведения будут подставлены из документа-основания счета-фактуры.

Предусмотрена возможность автоматической печати счетов-фактур при отгрузке товаров.

Для этих целей используется команда Печать – Настраиваемый комплект документов из формы документа "Расходная накладная" или из списка документов "Расходные накладные".
