Документ "Перемещение запасов"


Возможности и реквизиты данного документа отличаются в приложениях "1С:УНФ" и "1С:Розница". 
Если в справке напротив возможности или элемента интерфейса есть иконка, то описанная опция доступна только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Документ предназначен для оформления передачи запасов из одной структурной единицы в другую, а также списания запасов на внутренние расходы. 

В этой справке:

  • Заполнение документа
  • Перемещение зарезервированных товаров
  • Печать

 Важно! Отображение некоторых полей документа "Перемещение запасов" зависит от включенных возможностей программы. Активировать дополнительные возможности можно в разделе Настройки – Еще больше возможностей.

 

 

Заполнение документа

Документ "Перемещение запасов" может быть введен из списка документов "Перемещения запасов" (Раздел: "Склад").

Доступен ввод перемещения запасов на основании документов:

  • "Заказ на перемещение",
  • "Заказ покупателя",
  • "Приходная накладная",
  • "Приходный ордер",
  •  "Перемещение по ячейкам",
  • "Передача товаров между организациями",
  • "Авансовый отчет",
  •  "Заказ на производство",
  •  "Производство".

В зависимости от установленного вида операции с помощью документа "Перемещение запасов" может быть зафиксировано:

  • Перемещение,
  • Списание на расходы,
  •  Передача в эксплуатацию,
  •  Возврат из эксплуатации.

В документе необходимо заполнить следующие реквизиты: 

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ.
  •  Вид операции. Вид операции документа.
  • Склад. В этом поле необходимо указать структурную единицу, из которой будут списаны запасы.
  •  Ячейка. В этом поле необходимо указать ячейку, из которой будут списаны запасы.
  • Получатель. В этом поле необходимо указать структурную единицу, в которую будут оприходованы запасы.
  •  Ячейка (получатель). В этом поле необходимо указать ячейку, в которую будут оприходованы запасы.
  • Заказ покупателя. Заказ, под который будет осуществлен резерв или на который будут отнесены расходы.
  •  Счет затрат. В этом поле необходимо указать счет учета с типом "Расходы" или "Косвенные затраты" для определения направления списания запасов.
  •  Направление деятельности. Направление, на которое будут отнесены расходы. Имеет смысл для счетов типа "Расход". 

На закладке "Запасы" отражается список запасов, которые будут перемещены.

Табличная часть может быть заполнена с помощью подбора (кнопка "Подобрать" командной панели табличной части), сканированием штрихкода, методом ручного ввода, а также загрузкой из внешних источников (например, таблиц Excel).

Реквизиты табличной части:

  • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
  • Характеристика. При необходимости в данном поле указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если в настройках параметров учета указано ведение учета по характеристикам номенклатуры. Выбирается из справочника "Характеристики номенклатуры".
  • Партия. Указывается партия номенклатурной позиции. Поле доступно только в том случае, если в настройках параметров учета указано ведение учета по партиям номенклатуры. Выбирается из справочника "Партии".
  • Серия. Указывается серия номенклатурной позиции. Поле доступно только в том случае, если в настройках параметров учета указано ведение учета по сериям номенклатуры. Выбирается из справочника "Серии".
  • Ед. Единица измерения, в которой вводится количество номенклатуры.
  • Кол-во. Указывается количество единиц измерения номенклатурной позиции.
  • Резерв. Указывается количество единиц измерения номенклатурной позиции, передаваемой в резерв.
  • Заказ покупателя. Заказ, под который будет осуществлен резерв или на который будут отнесены расходы.

 

Перемещение зарезервированных товаров

Для перемещения товаров, зарезервированных под заказ покупателя, в качестве склада-получателя в документе "Перемещение запасов" указывается склад, на который должен быть перемещен товар, а в качестве склада-отправителя – склад, на котором товар был зарезервирован.

Табличное поле на закладке "Запасы" заполняется всеми позициями, зарезервированными по заказу покупателя на складе-отправителе, с учетом реального остатка на складе.

Для каждой позиции можно указать в колонке "Резерв" количество зарезервированного товара, предназначенного для перемещения.

Предусмотрена возможность оформления одного документа перемещения по нескольким заказам покупателя. В этом случае информация о каждом заказе указывается в колонке "Заказ покупателя" таблицы запасов.

Пример

Товар был зарезервирован под заказ покупателя на складе "Основной склад". Покупателю удобнее забрать часть товара с другого склада (например, "Склад временного хранения").

Тогда в документе "Перемещение запасов" в колонке "Заказ покупателя" нужно указать соответствующий заказ, а в колонке "Резерв" – количество единиц товара, которое покупатель хочет забрать из салона-магазина.

Отгрузка указанного количества зарезервированных по заказу товаров будет производиться со склада временного хранения. При проведении документа Расходная накладная отгруженный резерв по заказу покупателя автоматически снимется.

 

Печать

Из списка и формы документа "Перемещение запасов" можно быстро подготовить печатные документы:

  • М11 (Требование-накладная),
  • ТОРГ-13 (Накладная на внутреннее перемещение),
  • МХ-18 (Накладная на передачу готовой продукции),
  • 1-Т (товарно-транспортная накладная),
  • Транспортная накладная,
  • Перемещение запасов,
  • Бланк товарного наполнения (отправитель),
  • Бланк товарного наполнения (получатель),
  • печать этикеток,
  • печать ценников.

Печатные формы документов можно распечатать, сохранить как файл, отправить клиенту на электронную почту.

Для быстрой подготовки комплекта документов к перемещению предусмотрен помощник "Печать комплекта", доступный по ссылке "Настраиваемый комплект документов" в меню "Печать".

Состав комплекта настраивается с помощью флажков напротив документов, для каждого документа можно указать количество печатных копий.

На закладке "Дополнительно" по ссылке "Реквизиты печати" вы можете заполнить реквизиты для подстановки в печатные документы.

 

Признание расходов УСН

Существует две схемы признания расходов УСН при перемещении запасов с операцией "Списание на расходы" 
  • По факту проведения документа на указанное количество запасов, установкой флага «УСН».
  • По факту оприходования запасов флагом «Производство и пр. (УСН)» в приходной накладной.

Флаги являются взаимозаменяемыми, «Производство и пр. (УСН)» должен быть снят, если используется признание расходов в момент проведения документов производства с флагом «УСН».

Через команду "Шапка / табличная часть" в меню "Ещё", возможно указание флага "УСН" построчно в табличной части или в шапке документа.

По умолчанию, флаг «УСН» снят в документах производства. Для использования новой схемы признания расходов на производство, необходимо перейти в документе производства в «Настройки» - «Персональные настройки для всех документов» и выбрать «Да» в настройке «Значение флага УСН по умолчанию в документах производства и пр.».
