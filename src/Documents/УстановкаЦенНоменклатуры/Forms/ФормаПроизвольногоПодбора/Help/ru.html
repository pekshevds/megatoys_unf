"Подбор номенклатуры по отбору" для документа "Установка цен"


Форма предназначена для произвольного подбора номенклатуры в документ "Установка цен"

Форма основана на технологии "Компоновки данных" и позволяет настраивать произвольный отбор.

Левая верхняя таблица отвечает за элементы, которые могут быть добавлены в отбор.

Правая верхняя таблица содержит текущие элементы отбора. Они могут быть включены или выключены. При изменении этих элементов происходит автоматическое обновление табличной части подобранной номенклатуры.

Нижняя таблица содержит подобранную номенклатуру, которую можно перенести в документ одноименной кнопкой.


Для формы могут быть подключены произвольные макеты компоновки данных, например через расширение. Для подключения необходимо добавить в расширение документ "Установка цен", документу добавить макет  с именем по шаблону "ПодборСКД_???", где вместо "???" указать произвольное название. Синоним шаблона отвечает за имя в списке выбора схем компоновки на форме. В схеме должны быть обязательные поля "Номенклатура" и "Характеристика". Пример схемы можно взять из типового макета "ПодборСКД_Остатки".
