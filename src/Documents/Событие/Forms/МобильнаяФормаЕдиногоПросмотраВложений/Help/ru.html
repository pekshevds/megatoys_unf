Предварительный просмотр вложений письма и их изменение (мобильный клиент)


Можно предварительно посмотреть содержимое вложений письма. Если вызвать из черновика, то файлы можно добавлять, изменять и удалять. 

Для просмотра соответствующего файла следует выбрать его в списке справа, и тогда, если вложение доступно для предпросмотра, то его содержимое отобразится слева. 

Допустимые расширения файлов - bmp, emf, gif, tiff, ico, icon, jpg, jpeg, png, tif, wmf, svg, txt, ini, xml, htm, html, mxl.

Пример.

Для добавления и проверки файла можно открыть форму предпросмотра, добавить файл с устройства и в случае чего отредактировать его.
