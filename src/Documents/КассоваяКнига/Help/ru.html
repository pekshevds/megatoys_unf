Документ "Кассовая книга"

Документ "Кассовая книга" (КК) предназначен для формирования и печати листов кассовой книги организации.

В этой справке:

  • Принципы работы
  • Заполнение документа
  • Печать кассовой книги
  • Контроль кассовой книги
  • Кассовая книга платежного агента


Принципы работы

Кассовая книга представляет собой регистр, в котором отражаются все операции с наличными денежными средствами. В кассовой книге регистрируются все кассовые операции немедленно после их совершения с указанием номера кассового документа, на основании которого произведён приём или выдача наличных денег, фамилии лица, сдавшего или получившего деньги, и суммы операции.

Кассовая книга формируется по каждой отдельной операционной кассе организации.

Суммы безналичной оплаты (эквайринг, оплата через онлайн-банк и др. виды безналичных расчетов), которые проходят через онлайн-кассу, в кассовую книгу не попадают.

Важно! Заполнять листы кассовой книги по конкретной организации невозможно при использовании настройки учета "Учет по компании в целом". Так как в данном случае движения денежных средств формируются без аналитики по организациям.

Чтобы сформировать кассовую книгу необходимо перейти в раздел Деньги - Касса и нажать на ссылку "Кассовая книга".

Откроется список документов "Лист кассовой книги". Для удобства работы пользователь может отфильтровать список листов кассовой книги, установив значение отбора в поле "Организация" и "Касса" в правой панели списка.

Контроль полноты кассовой книги осуществляется с помощью отчета "Контроль кассовой книги".

  Перейти к справке "Контроль кассовой книги"


Заполнение документа

По окончании операционного дня пользователь создает новый документ "Лист кассовой книги". Новый лист кассовой книги создается с помощью кнопки "Создать" в командной панели списка документов.

За день может быть введен только один лист для каждой кассы и организации.

В документе "Лист кассовой книги" указываются следующие реквизиты:

  • Касса – касса, место фактического хранения и движения наличных денежных средств.
  • Организация – организация, от имени которой оформляется документ.
  • Тип листа – определяет, какая печатная форма будет использоваться для документа. Листы бывают следующих типов:
    • Обычный – без вывода итогового количества листов;
    • Последний в месяце – вывод итогового количества листов за месяц;
    • Последний в году – вывод итогового количества листов за год.

Поле "Организация" может быть заполнено по умолчанию. Значение поля определяется значением быстрого отбора из списка листов кассовой книги, значением по умолчанию для текущего пользователя или единственным в информационной базе значением организации.


Заполнение табличной части

Табличная часть документа содержит список кассовых операций. По кнопке "Заполнить" таблица заполняется документами "Поступление в кассу" и "Расход из кассы" за указанную в шапке документа дату.

Колонка "Номер листа" выводится информативно и пересчитывается при проведении документа.

Поле "Корреспондирующий счет" заполняется если включена синхронизация с "1С:Бухгалтетией". Значение корреспондирующего счета подставляется из статьи движения денежных средств, указанной в кассовом документе.


Печать кассовой книги

Предусмотрена возможность печати листов и обложки кассовой книги на определенную дату по каждой кассе организации.

Печать листов кассовой книги осуществляется из выпадающего меню "Печать" формы документа или формы списка документов.

Печать кассовой книги осуществляется на листах формата А4.

Печать титульного листа и последней страницы осуществляется по команде "Титульный лист и последняя страница" выпадающего списка "Печать".

Реализовано две печатных формы – "лист кассовой книги" и "обложка" (титульный лист + последний лист "прошито пронумеровано").

По команде "Настраиваемый комплект документов" выполняется пакетная печать. В этом случае обложка формируется для листа с типом "последний в году".

Из списка документов "Кассовая книга" можно распечатать сразу несколько листов кассовой книги, выделив их с помощью клавиш CTRL или Shift.

 

Контроль кассовой книги

Контроль полноты кассовой книги осуществляется с помощью отчета "Контроль кассовой книги". Перейти к отчету можно из списка документов "Кассовая книга", а также из списка отчетов раздела "Деньги".

В отчет выводится список документов, которые:

  • не вошли в кассовую книгу по каким-то причинам;
  • были изменены после включения в кассовую книгу;
  • документы со сбитой нумерацией.

Для включения пропущенных документов в кассовую книгу необходимо перезаполнить документ "Лист кассовой книги" за дату не вошедшего в книгу кассового документа.

После внесения изменений потребуется распечатать новый лист кассовой книги, а также перепечатать листы кассовой книги за все последующие даты.

Для исправления нумерации листов кассовой книги перейдите из отчета по ссылке "Перенумеровать" и выполните обновление нумерации.

Рекомендуем использовать отчет "Контроль кассовой книги" перед печатью листов кассовой книги, чтобы избежать ошибок и исключить повторную печать.

 

Кассовая книга платежного агента

Предусмотрена возможность формирования кассовой книги от лица платежного агента. Кассовая книга составляется отдельно для каждого агентского договора.

При включенной настройке учета "Агентские платежи и разделение выручки"  в форме документа "Лист кассовой книги" появляется реквизит "Договор", позволяющий формировать записи по каждому агентскому договору.
