Документ "Пересортица запасов"


Возможности и реквизиты данного документа отличаются в приложениях "1С:УНФ" и "1С:Розница". 
Если в справке напротив возможности или элемента интерфейса есть иконка, то описанная опция доступна только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Документ "Пересортица запасов"  предназначен для отражения пересортицы. С помощью документа можно быстро скорректировать остатки товаров.

Документ может быть оформлен на основании проведенной инвентаризации или как самостоятельный документ.

В этой справке:

  • Заполнение документа
  • Пересортица по данным инвентаризации

 

Заполнение документа

При оформлении пересортицы необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ.
  • Склад, на котором обнаружена пересортица. Если пересортица произошла с товарами, которые находятся на разных складах, то предварительно потребуется переместить товары на один склад.
  •  Корреспонденция списания. В этом поле необходимо указать счет учета с типом Прочие расходы или Прочие оборотные активы для определения направления списания запасов.
  •  Корреспонденция оприходования. В этом поле необходимо указать счет учета с типом Прочие доходы или Капитал для определения источника оприходования запасов.
  • Оприходовать товары по себестоимости списания – отвечает за то, по какой цене будут оприходованы товары. Если флажок не установлен, пользователь самостоятельно может указать корреспонденцию списания/оприходования и стоимость, по которой запас будет оприходован. В противном случае поля корреспонденций и стоимости будут скрыты, а товары будут оприходованы по себестоимости списания.
  • Основание. Выбор или просмотр документа инвентаризации, на основании которого производится списание.
  • Комментарий. Произвольный комментарий к документу.

На закладке "Запасы" вносятся данные об излишках и недостачах. Для быстрого заполнения табличной части предусмотрены: группа фильтров "Номенклатура в" и команды в подменю "Заполнить", расположенные над табличной частью.

В таблице доступны для редактирования следующие колонки:

  • Номенклатура (списание). В данное поле вводится элемент справочника номенклатуры, который будет списан.
  • Характеристика. При необходимости в данном поле указывается характеристика списываемой номенклатуры. Поле доступно только в том случае, если включена опция ведение учета по характеристикам номенклатуры. Выбирается из справочника "Характеристики номенклатуры".
  • Партия. Указывается партия списываемой номенклатурной позиции. Поле доступно только в том случае, если включена опция ведение учета по партиям номенклатуры. Выбирается из справочника "Партии".
  • Серийные номера. По двойному щелчку в этом поле будет открыт диалог подбора серийных номеров к списанию. Возможность доступна, если включена опция учета серийных номеров.
  • Страна и Номер ГТД. В этих полях можно ввести страну и номер ГТД импортного товара. Возможность доступна, если включена опция ведения учета по ГТД.
  • Ед. Единица измерения, в которой вводится количество списываемой номенклатуры.
  • Количество. Указывается количество единиц измерения, которое будет списано и оприходовано.

Аналогичные колонки, характеризующие запас, заполняются для приходуемой номенклатуры, которая указывается в колонке "Номенклатура (оприходование)".

Указать цену и сумму номенклатуры, по которым она будет оприходована, можно только если в документе на закладке "Главное" не установлен флаг "Приходовать товары по себестоимости списания".

При проведении документа "Пересортица запасов" будут сформированы проводки по списанию и оприходованию запасов.

Из документа доступна печать формы "Акт о пересортице запасов".

 

Пересортица по данным инвентаризации

При создании документа "Пересортица запасов" на основании инвентаризации, таблица "Запасы" будет заполнена автоматически в соответствии с данными об излишках и недостачах, обнаруженных в результате проведенной инвентаризации.

Позиции товаров, по которым выявлены расхождения, необходимо сопоставить между собой.

Сделать это можно автоматически с помощью команды "Автоматически сопоставить все пересортицы". Также можно вручную выделить 2 строки (удерживая клавишу Ctrl) и нажать на кнопку "Сопоставить 2 строки".

Если количество в сопоставляемых строках совпадает, то вместо двух строк будет сформирована одна.
