Документ "Доверенность"

Документ предназначен для оформления доверенностей на получение товаров.

Доверенность на получение товаров может быть оформлена из раздела Закупки. Основанием для оформления доверенности на получение товаров может быть "Заказ поставщику", "Счет на оплату (полученный)".

Для оформления доверенности необходимо указать "Доверенное лицо" (из справочника "Физические лица") – получателя товарно-материальных ценностей.

В шапке документа необходимо заполнить "Организацию" (если включено ведение учета по нескольким организациям), от которой выписывается доверенность, и "Банковский счет" организации.

Эти данные отображаются в печатной форме доверенности.

На закладке "Печать" должны быть заполнены:

  • Паспортные данные. Подставляются из карточки физического лица.
  • Поставщик, для которого выписывается доверенность. Выбирается из справочника "Контрагенты".
  • Договор контрагента. Выбирается из справочника "Договоры контрагентов".
  • Дата действия. Срок действия доверенности.
  • На получение от. Полное наименование контрагента, которое отображается в печатной форме документа
  • По документу. Полное наименование документа основания, которое отображается в печатной форме документа

На закладке "Запасы" заполняется перечень товаров к получению по доверенности:

  • Наименование товара, получаемого по доверенности.

  • Ед. – единица измерения товара.

  • Количество единиц к получению по доверенности.
