Форма "Калькуляция заказа"

Форма предназначена для корректировки состава спецификаций, расчета предварительной стоимости и себестоимости заказа (калькуляции заказа).

Переход к калькуляции выполняется из документов "Заказ покупателя" и "Заказ-наряд".  Расчет выполняется автоматически при открытии формы "Калькуляция".

В этой справке:

  • Выполнение калькуляции
  • Варианты калькуляции
  • Просмотр и печать итоговой калькуляции

 Важно! Отображение некоторых полей формы "Калькуляция" зависит от включенных возможностей программы. Активировать дополнительные возможности можно в разделе Настройки – Еще больше возможностей.


        


 

Выполнение калькуляции

Работа с калькуляцией может выполняться в режимах:

  • с отображением данных по себестоимости и прибыли
  • с отображением только данных по количеству и сумме продажи.

Переключение между режимами выполняется помощью кнопки  в форме "Калькуляция".

При включенном отображении данных по себестоимости пользователь может выбирать, по какой цене рассчитывать себестоимость заказа:

  • по последней цене закупки, по ценам конкретного поставщика,
  • по одному из заведенных в базе видов цен.

Для пересчета калькуляции следует использовать кнопку "Обновить калькуляцию".

Калькуляцию можно загрузить из табличного документа или расчетной программы с помощью универсальной загрузки данных из внешних источников, вызываемой по кнопке .

Расчет плановой себестоимости заказа, состоящего из изделий и полуфабрикатов, производится на основании спецификаций.

При заполнении калькуляции состав спецификации можно изменять, модифицированная копия спецификации сохраняется с привязкой к конкретному заказу.

 Перейти к справке "Спецификации"

 Перейти к статье "Состав продукции и технология ее выпуска"

 

Кроме материального состава в калькуляции можно учесть стоимость операций, затраты на зарплату сотрудников и прочие расходы (например, доставку).

Если необходимо учесть разовый расход по заказу, его можно добавить с помощью кнопки "Добавить расход".

Для постоянных расходов рекомендуется создать шаблон калькуляции.

 

Варианты калькуляции

Создавать варианты калькуляции возможно по всем коммерческим предложениям заказа покупателя. В программе должна быть включена возможность ввода вариантов коммерческого предложения.

Чтобы создать калькуляцию для варианта коммерческого предложения, необходимо выбрать вариант в одноименном поле в форме "Калькуляция".

Такое решение позволит сравнить себестоимость, выручку и прибыль нескольких вариантов коммерческих предложений и выбрать наиболее выгодный способ выполнения заказа.

Просмотр и печать итоговой калькуляции

Готовая калькуляция сохраняется в заказе покупателя и доступна для просмотра и редактирования по гиперссылке "Калькуляция заказа".

Из формы калькуляции можно распечатать расчет по кнопке .

Сравнить плановый расчет с фактическими доходами и расходами по заказу можно в отчете "Сводный отчет по заказу", открыть который можно из формы заказа по ссылке "Сводно о заказе".

В отчете отображается полная информация о заказе: данные заказчика, сумма заказа, все примененные скидки и наценки, товарный состав заказа, плановые и итоговые доходы и расходы.

Сводная информация по заказу покажет прибыль, рентабельность и текущий процент наценки по заказу.

Опираясь на  полученные данные, вы сможете объективно принимать решения о необходимости наценки или возможности предоставления скидки клиенту.
