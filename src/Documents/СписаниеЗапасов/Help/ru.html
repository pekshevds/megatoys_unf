Документ "Списание запасов"


Возможности и реквизиты данного документа отличаются в приложениях "1С:УНФ" и "1С:Розница". 
Если в справке напротив возможности или элемента интерфейса есть иконка, то описанная опция доступна только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Документ предназначен для оформления факта списания материальных ценностей, имеющихся в наличии на складе или введенных в эксплуатацию на подразделении.

Документ может быть оформлен на основании проведенной инвентаризации или как свободный документ в случае произвольного списания запасов.

Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ.
  • Склад. В этом поле необходимо указать структурную единицу - склад или  производственное подразделение, из которого будут списаны запасы.
  • Комментарий. Произвольный комментарий к документу.
  •  Ячейка. В этом поле необходимо указать ячейку склада, с которой будут списаны запасы.
  •  Корреспонденция. В этом поле необходимо указать счет учета с типом  "Прочие расходы" или "Прочие оборотные активы" для определения направления списания запасов.

Табличная часть "Запасы":

  • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
  • Характеристика. При необходимости в данном поле указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если в настройках учета указано ведение учета по характеристикам номенклатуры. Выбирается из справочника "Характеристики номенклатуры".
  • Партия. Указывается партия номенклатурной позиции. Поле доступно только в том случае, если в настройках учета указано ведение учета по партиям номенклатуры. Выбирается из справочника "Партии".
  • Серия.  Указывается серия номенклатурной позиции. Поле доступно только в том случае, если в настройке учета указано ведение учета по сериям номенклатуры. Выбирается из справочника "Серии номенклатуры".
  • Ед. Единица измерения, в которой вводится количество номенклатуры. 
  • Кол-во. Указывается количество единиц измерения номенклатурной позиции.
  • Страна происхождения номенклатурной позиции. Выбирается из справочника "Страны мира".

  • Номер ГТД. Указывается номер грузовой таможенной декларации. Выбирается из справочника "Номера ГТД". Возможность автоматического подбора номера ГТД включается/выключаются в настройках программы (см. раздел Закупки).

По ссылке "Основание"  можно выбрать или просмотреть документ инвентаризации, на основании которого производится списание.


Признание расходов УСН

Существует две схемы признания расходов УСН при перемещении запасов 

  • По факту проведения документа производства на указанное количество запасов, установкой флага «УСН».
  • По факту оприходования запасов флагом «Производство и пр. (УСН)» в приходной накладной.

Флаги являются взаимозаменяемыми, «Производство и пр. (УСН)» должен быть снят, если используется признание расходов в момент проведения документов производства с флагом «УСН».

По умолчанию, флаг «УСН» снят в документах производства. Для использования новой схемы признания расходов на производство, необходимо перейти в документе производства в «Настройки» - «Персональные настройки для всех документов» и выбрать «Да» в настройке «Значение флага УСН по умолчанию в документах производства и пр.».
