Документ "Чек ККМ на возврат"

Документ предназначен для хранения информации по отражению в учете операции по возвратам реализованных запасов, услуг и работ от розничного покупателя.

Карточка "Чека ККМ на возврат" идентична документу "Чек ККМ".

Чек возврата может быть введен на основании  документа "Чек ККМ", при этом табличное поле "Товары и услуги" будет заполнено всем списком товаров документа-основания.

Ели возвращается определенный товар из списка, продавец удаляет лишние позиции, в таблице "Товары и услуги" должны остаться только товары, которые вернул покупатель.

Остальные вкладки документа заполняются теми же данными, что и в документе "Чек ККМ".

 

Возврат в РМК

Чек ККМ на возврат может быть сформирован из "Рабочего места кассира (РМК)".

Для оформления возврата от покупателя в меню "Режим" необходимо выбрать операцию "Возврат продажи" и указать чек для возврата, который можно найти по номеру, времени или сумме.

Также для подбора чека продажи можно отсканировать его код.

После подбора нужного чека, товары из него будут добавлены в РМК. Кассиру останется проверить данные и пробить чек с помощью одноименной кнопки.

При возврате по чеку продажи, суммы автоматически заполняются по чеку продажи.

Если была продажа только за наличные, то нельзя вернуть оплату на карту и наоборот. При продаже со смешанной оплатой на карту нельзя вернуть сумму большую, чем была в чеке продажи.

При оформлении возврата покупателя без чека кассиру, требуется вручную добавить товары и способ оплаты. После пробития чека на ККТ автоматически формируется документ "Чек ККМ на возврат".
