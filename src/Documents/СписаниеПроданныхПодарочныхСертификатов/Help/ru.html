Документ "Списание проданных сертификатов"


Возможности и реквизиты данного документа отличаются в приложениях "1С:УНФ" и "1С:Розница". 
Если в справке напротив возможности или элемента интерфейса есть иконка, то описанная опция доступна только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Документ предназначен для списания реализованных подарочных сертификатов.

Для сертификатов с ограниченным сроком действия возможна ситуация, когда держатель не воспользовался сертификатом в указанный в сертификате срок. В этом случае потребуется оформить списание сертификата, а полученный по нему аванс учесть как доход организации.

Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ.
  •  Корреспонденция. В этом поле необходимо указать счет учета с типом  "Прочие доходы".
  • Комментарий. Произвольный комментарий к документу.

Табличная часть "Подарочные сертификаты":

  • Подарочный сертификат. В данное поле вводится вид сертификата.
  • Номер сертификата. При использовании серийных номеров в данном поле указывается номер сертификата.
  • Сумма. Поле заполняется автоматически исходя из остатка по сертификату на дату документа.

Документ можно автоматически заполнить сертификатами с истекшим сроком действия, либо добавить любые проданные сертификаты вручную. Команды автоматического заполнения расположены в подменю "Заполнить":

  • заполнение просроченными сертификатами;
  • перезаполнение суммы по остаткам, внесенных в документ сертификатов.

При проведении документа, выданный по сертификатам аванс списывается на доходы (согласно указанной настройке), а номинал аннулируется.
