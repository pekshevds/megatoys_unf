Документ "Счет-фактура (полученный)"


Возможности и реквизиты данного документа отличаются в приложениях "1С:УНФ" и "1С:Розница". 
Если в справке напротив поля или элемента интерфейса есть иконка, то элемент доступен только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Документ предназначен для регистрации полученных счетов-фактур от поставщиков, а также для учета номеров грузовых таможенных деклараций.

Счет-фактура может быть выписан:

  • На поступление,
  • На аванс,
  • На суммовые разницы.

 

Важно! Отображение некоторых полей документа зависит от включенных возможностей программы. Активировать дополнительные возможности можно в разделе Настройки – Еще больше возможностей.

 

 

Оформление счета-фактуры

Документ "Счет-фактура (полученный)" оформляется на основании документов:

  • "Заказ поставщику",
  • "Приходная накладная".
  • "Дополнительные расходы",
  • "Авансовый отчет",
  • "Расход со счета",
  • "Расход из кассы",
  • "Отчет комиссионера" и "Отчет комитенту" - при использовании комиссионных продаж.
  •  "Отчет пеработчика".

При создании нового счета-фактуры в зависимости от документа-основания определяется вид документа: "На поступление", "На аванс", "На суммовые разницы", и автоматически заполняется в поле "Операция".

Из списка документов "Счета-фактуры" по умолчанию создается счет-фактура "На поступление".

Создать счет-фактуру можно непосредственно из подвала документа-основания (например, приходной накладной) по гиперссылке "Создать".

В счет-фактуру переносятся сведения из соответствующего документа-основания.

Состав реквизитов счета-фактуры:

  • Организация, в которой регистрируется счет-фактура. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Вид операции. Указывается вид операции документа.
  • Основание. Указывается документ-основание получения счета-фактуры.
  • Контрагент. Указывается поставщик, от которого выписан счет-фактура. Выбирается из справочника "Контрагенты".
  • Договор поставщика, от которого выписан счет-фактура. Выбирается из справочника "Договоры контрагентов".
  • Вх. номер Указывается номер входящего документа, присвоенный поставщиком.
  • от.  Указывается дата входящего документа.
  • Комментарий. Указывается любая дополнительная информация к документу.
