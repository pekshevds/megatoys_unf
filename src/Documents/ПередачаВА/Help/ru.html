Документ "Продажа имущества"


Возможности программы отличаются в зависимости от используемой конфигурации: "1С:УНФ" или "1С:Розница". 
Документ "Продажа имущества" доступен только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Документ предназначен для оформления продажи имущества. 

 Список документов хранится в журнале документов по учету имущества.

 При проведении документа выполняется несколько операций:

  • доначисление амортизации за месяц выбытия;
  • списание имущества с учета;
  • отражение расчетов с покупателем.

В документе необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, от которой производится продажа имущества. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Контрагент.  В данном реквизите указывается информация о покупателе. Выбирается из справочника "Контрагенты".
  • Договор. В данном реквизите указывается договор контрагента, в рамках которого оформляется документ. Выбирается из справочника "Договоры контрагентов".
  • Комментарий. Указывается комментарий к документу.
  • Подпись кассира - поле доступно для вида операции "Заказ-наряд". Если поле пустое, то при печати чека ККМ из формы документа, выводится ФИО по данным автора документа. Если поле заполнено, то при печати чека ККМ из формы документа, выводится ФИО по данным подписи кассира, которое выбрано в поле.

На форме документа расположена гиперссылка "Редактировать цены и валюту", по которой вызывается диалог редактирования валюты данного документа.

Табличная часть "Имущество":

  • Имущество, передаваемое покупателю. Выбирается из справочника "Имущество".

  • Начальная стоимость имущества.
  • Остаточная стоимость имущества. Начальная стоимость за вычетом амортизации всего.
  • Амортизация всего. Сумма амортизации, начисленная за весь период, до текущего месяца.
  • Амортизация за тек. месяц. Сумма амортизации, начисленная за текущий месяц.
  • Сумма реализации имущества, с налогами согласно включенным флагам.
  • Ставка НДС - ставка налога на добавленную стоимость. Выбирается из справочника "Ставки НДС".

  • Сумма НДС - сумма налога на добавленную стоимость по данной строке.
  • Всего - поле, автоматически заполняемое суммарной стоимостью имущества.

Табличная часть "Зачет предоплаты" заполняется сведениями по зачету авансов от покупателя. 

На форме документа расположена гиперссылка "Редактировать зачет предоплаты", по которой вызывается диалог редактирования табличной части "Зачет предоплаты".

Реквизиты табличной части "Зачет предоплаты":

  • Документ - документ поступления аванса: "Поступление в кассу" или "Поступление на счет".
  • Сумма расчетов - сумма в валюте расчетов.
  • Курс - курс валюты расчетов. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
  • Кратность - кратность валюты расчетов. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
  • Сумма платежа - указывается в валюте документа. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
