Документ "Положение о премировании"

Документ используется для ввод в действие правил расчета премий продавцам по результатам продаж.

Документ формирует фиксирует правила, которые будут использоваться для расчета премии продавцам согласно объему продаж.

В документе заполняются:

  • Период действия - "с - по" - период действия положения о премировании, окончание периода может быть не заполнено.

  • Вариант использования положения:

  • "Индивидуальное для магазина" - для каждого магазина выбираются собственные правила

  • "Одно для выбранных магазинов" - для списка магазинов выбираются общий список правил

  • "Одно для всех магазинов" - для всех магазинов выбираются общий список правил

  • Магазины - список магазинов положения
  • Правила расчета премий - список действующий в положении правил
