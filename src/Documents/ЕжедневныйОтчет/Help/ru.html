Документ "Ежедневный отчет"

Документ предназначен для регистрации рабочего времени сотрудников в течении дня. Список документов открывается по ссылке Персонал - Ежедневные отчеты.  

В документе необходимо заполнить следующие реквизиты:

  • Рабочая смена, для которой будет вводиться документ ежедневный отчет. Выбирается из списка справочника рабочие смены.
  • Отчет за день. Указывается дата за которую будут фиксироваться данные рабочего дня.

  • Список сотрудников. В табличной части указываются сотрудники, время начала и окончания работы. Реквизиты табличной части:

    • Сотрудник - сотрудник смены. Выбирается из справочника сотрудники. 

    • Начало работы - время начала работы.

    • Окончание работы - время окончания работы.

    • За смену (ч) - время работы.


  • Комментарий. Комментарий документа
  • Ответственный. Ответственный за документ.

В табличной части возможно указание нескольких периодов для одного сотрудника. Например рабочее время до обеденного перерыва и после.

 
Документ может быть заполнен с помощью подключаемого оборудования - сканера штрихкодов и считывателя магнитных карт. Для этого оборудование должно быть подключено, а у сотрудников в их карточках указаны штрихкоды и номера магнитных карт.

Если через оборудование будет получен новый код, то документ предложит возможность привязать этот код сотруднику не покидая форму документа. Операции добавления новых кодов не сохраняются при закрытии формы документа и будут очищены.

При добавлении сотрудника в табличную часть через подключаемое оборудование будет установлено новое время начала работы, если в табличной части уже есть этот сотрудник с началом работы и пустым окончанием, то окончание работы будет заполнено. Если уже есть сотрудник с заполненным окончанием, то будет добавлена новая строка с началом работы.
