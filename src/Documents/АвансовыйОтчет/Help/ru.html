Документ "Авансовый отчет"


Возможности и реквизиты данного документа отличаются в приложениях "1С:УНФ" и "1С:Розница". 
Если в справке напротив возможности или элемента интерфейса есть иконка, то описанная опция доступна только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Документ предназначен для отражения в учете операций по расходованию подотчетным лицом выданных ему наличных или безналичных денежных средств.

В этой справке:

  • Заполнение документа
    • Выданные авансы
    • Запасы
    • Услуги
    • Оплаты
    • Дополнительно
  • Ввод документа на основании
  • Расчеты с подотчетником в валюте

 Важно! Отображение некоторых полей документа зависит от включенных возможностей программы. Активировать дополнительные возможности можно в разделе Настройки – Еще больше возможностей.

 

 

Заполнение документа

Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, для которой оформляется авансовый отчет. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Сотрудник. В данном реквизите указывается подотчетник. Выбирается из справочника "Сотрудники".
  • Комментарий. Указывается комментарий к документу.

Документ имеет несколько табличных частей, которые располагаются на соответствующих закладках:

  • Выданные авансы.  На закладке отображается информация о денежных средствах, полученных подотчетным лицом.
  • Запасы. На закладке заполняются данные о закупленных подотчетником запасах.
  • Услуги. На закладке заполняются данные об оплаченных подотчетником услугах.
  • Оплаты. На закладке заполняются данные о выплаченных подотчетником денежных средствах поставщикам заполняются на закладке "Оплаты".

Рассмотрим подробнее заполнение каждой из закладок:

 

Выданные авансы

На закладке отображается информация о денежных средствах, полученных подотчетным лицом.

Реквизиты табличной части:

  • Документ выдачи денежных средств подотчетному лицу. Выбирается документ "Расход из кассы" или "Расчет со счета".
  • Сумма - израсходованная сумма по документу выдачи.

 

Запасы

На закладке заполняются данные о закупленных подотчетником запасах.

Реквизиты табличной части:

  • Номенклатура - приобретенная подотчетником. Выбирается из справочника "Номенклатура" с типом "Запас".
  • Характеристика - при необходимости в данном поле указывается характеристика номенклатуры. Выбирается из справочника "Характеристики номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по характеристикам номенклатуры. Реквизит не обязателен для заполнения.
  • Партия - партия номенклатуры, по которой принимается номенклатурная позиция. Выбирается из справочника "Партии номенклатуры". Реквизит не обязателен для заполнения.
  • Ед. - единица измерения номенклатурной позиции. Выбирается из справочника "Классификатор единиц измерения" или "Единицы измерения".
  • Количество - количество принимаемой номенклатуры, за которую отчитывается подотчетник.
  • Цена - цена, по которой номенклатурная позиция была куплена подотчетником.
  • Сумма - стоимость номенклатуры по данной строке, с налогами согласно включенным флажкам.
  • Ставка НДС - ставка налога на добавленную стоимость. Выбирается из справочника "Ставки НДС".
  • Сумма НДС - сумма налога на добавленную стоимость по данной строке.
  • Всего - поле, автоматически заполняемое суммарной стоимостью запасов.
  • Заказ покупателя - заказ покупателя, в рамках которого приобретены запасы. Выбирается документ "Заказ покупателя". Реквизит не обязателен для заполнения.
  • Структурная единица - склад, на который приходуются запасы. Выбирается из справочника "Структурные единицы" с типом "Склад".
  •  Ячейка склада, на который приходуются запасы. Выбирается из справочника "Ячейки". Реквизит не обязателен для заполнения.
  • Дата вх. док - дата входящего документа.
  • Номер вх. док - номер входящего документа.
  • Содержание - представление запаса в печатной форме. Чтобы реквизит стал видимым, необходимо выполнить "Все действия" - "Изменить форму" и установить флаг напротив реквизита.
  • Статья движения денежных средств для кассового метода учета доходов и расходов. Выбирается из справочника "Статьи движения денежных средств". Реквизит отображается на форме, если в настройках параметров учета включено использование кассового метода учета доходов и расходов. Реквизит не обязателен для заполнения.

 

Услуги

На закладке заполняются данные об оплаченных подотчетником услугах.

Реквизиты табличной части:

  • Номенклатура - услуга, оплаченная подотчетником. Выбирается из справочника "Номенклатура" с типом "Услуга".
  • Ед. - единица измерения номенклатурной позиции. Выбирается из справочника "Классификатор единиц измерения" или "Единицы измерения".
  • Количество - количество оплаченных услуг, за которое отчитывается подотчетник.
  • Цена - цена услуги.
  • Сумма - стоимость номенклатуры по данной строке, с налогами согласно включенным флажкам.
  • Ставка НДС - ставка налога на добавленную стоимость. Выбирается из справочника "Ставки НДС".
  • Сумма НДС - сумма налога на добавленную стоимость по данной строке.
  • Всего - поле, автоматически заполняемое суммарной стоимостью услуг.
  • Заказ покупателя - заказ покупателя, в рамках которого были понесены расходы. Выбирается документ "Заказ покупателя". Реквизит не обязателен для заполнения.
  • Подразделение, на которое относится расход. Выбирается из справочника "Структурные единицы" с типом "Административно-управленческое подразделение" или "Производство". Реквизит не обязателен для заполнения.
  •  Направление деятельности, на которое относится расход. Выбирается из справочника "Направления деятельности". Реквизит не обязателен для заполнения.
  • Дата вх. док - дата входящего документа.
  • Номер вх. док - номер входящего документа.
  • Содержание - представление услуги в печатной форме. Чтобы реквизит стал видимым, необходимо выполнить "Все действия" - "Изменить форму" и установить флаг напротив реквизита.
  • Статья движения денежных средств для кассового метода учета доходов и расходов. Выбирается из справочника "Статьи движения денежных средств". Реквизит отображается на форме, если в настройках параметров учета включено использование кассового метода учета доходов и расходов. Реквизит не обязателен для заполнения.

 

Оплаты

Данные о выплаченных подотчетником денежных средствах поставщикам заполняются на закладке "Оплаты".

Реквизиты табличной части:

  • Контрагент - поставщик, которому совершил оплату подотчетник. Выбирается из справочника "Контрагенты".
  • Договор - договор с контрагентом, по которому совершил оплату подотчетник. Выбирается из справочника "Договоры".
  • Аванс - признак аванса платежа.
  • Документ - документ по которому осуществляется оплата.
  • Сумма расчетов - сумма оплаты. Указывается в валюте расчетов по договору с контрагентом.
  •  Курс - курс валюты расчетов по договору с контрагентом. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
  •  Кратность - кратность валюты расчетов по договору с контрагентом. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
  •  Сумма платежа - сумма оплаты. Указывается в валюте кассы. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
  • Заказ - заказ поставщику. Указывается в случае осуществления платежа по определенному заказу. Выбирается документ "Заказ поставщику". Реквизит не обязателен для заполнения.
  • Дата вх. док - дата входящего документа.
  • Номер вх. док - номер входящего документа.
  • Статья движения денежных средств для кассового метода учета доходов и расходов. Выбирается из справочника "Статьи движения денежных средств". Реквизит отображается на форме, если в настройках параметров учета включено использование кассового метода учета доходов и расходов. Реквизит не обязателен для заполнения.

 

Дополнительно

В документе "Авансовый отчет" предусмотрена возможность автоматического создания счетов фактур в строках табличных частей "Запасы" и "Услуги".

Флажок "Использовать счет-фактуру в табличной части" на закладке "Дополнительно"  позволяет на закладках "Запасы" и "Услуги" указывать различных поставщиков в таблично части (заполняются построчно).

В результате проведения документа будут сформированы счета-фактура (полученные) для каждого из поставщиков  по строкам "Авансового отчета". При закупке нескольких товаров у одного поставщика единоразово, будут сформирован один документ "Счет-фактура (полученный)".

Также на закладке "Дополнительно" можно:

  • указать Назначение документа и  Проект,

  • заполнить реквизиты печати,

  • добавить свой реквизит.

 

Ввод документа на основании

Авансовый отчет может быть введен на основании документов:
  • Расход со счета;
  • Расход из кассы;
  • Приходный ордер.

 

 Расчеты с подотчетником в валюте

Документ может оформляться в той  валюте, которая определена для сотрудника при выдаче ему денежных средств (документ "Расход из кассы" или "Расход со счета") как валюта подотчетного лица.

На форме документа расположена гиперссылка "Редактировать цены и валюту", по которой вызывается диалог редактирования валюты данного документа.
