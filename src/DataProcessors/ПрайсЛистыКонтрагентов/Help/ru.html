Обработка"Прайс-лист контрагентов (поставщиков)"

Обработка предназначена для работы с входящими ценами (цены поставщиков).

В ней выводится информация о текущих ценах, зарегистрированных при оформлении документов поставки. Для отображения можно выбрать все виды цен, которые зарегистрированы для различных поставщиков.

Обработка позволит сравнить цены различных поставщиков между собой и быстро найти максимальную/минимальную цену, а также отобрать все цены, соответствующие заданному условию.

В этой справке:

  • Внешний вид;
  • Настройки прайс-листа;
    • Цены;
    • Колонки;
    • Фильтры;
    • Оформление;
    • Дополнительная информация.

 

 Важно! Отображение некоторых реквизитов обработки "Прайс-лист поставщиков" зависит от включенных возможностей программы. Активировать дополнительные возможности можно в разделе Настройки – Еще больше возможностей.

 

 

Внешний вид

Работа с прайс-листами компании и прайс-листами поставщиков реализована в едином интерфейсе.

Прайс-лист поставщиков представляет собой таблицу с перечнем товаров и ценами на них.

Цены в прайс-листе заполняются по выбранным в прайс-листе видам цен. Для отображения можно выбрать все виды цен, которые зарегистрированы для различных поставщиков.

 Перейти к справке "Виды цен контрагентов"

 

Настройки отображения прайс-листа поставщиков выполняются в правой панели.

ВАЖНО! Обработка "Прайс-листы контрагентов" не хранит информацию о товарах, ценах и способах их отображения, это только инструмент для построения прайс-листа. Все настройки хранятся в справочнике "Прайс-листы контрагентов". 

Прайс-листы можно сохранять в виде файлов на компьютер или отправлять непосредственно из программы с помощью кнопок на панели команд.

 

Настройки прайс-листа

Настройки прайс-листов собраны в группы:

  • Цены,
  • Колонки,
  • Фильтры,
  • Оформление,
  • Прочее.

Поле "Рабочий прайс-лист поставщиков" позволяет быстро выбрать (переключить) прайс-лист для работы.

Напротив поля располагаются кнопки "Создать" и "Скопировать" с помощью которых можно быстро подготовить новые прайс-листы.

Список всех прайс-листов контрагентов находится по ссылке "Показать все" в выпадающем меню поля "Рабочий прайс-лист".

В поле "Иерархия"можно выбрать, по какому справочнику построить дерево прайс-листа:

  • Иерархия групп номенклатуры. Отчет будет сформирован по папкам номенклатуры.

  • Иерархия ценовых групп. Отчет будет сформирован по справочнику "Ценовые группы".

  • Иерархия категорий номенклатуры. Отчет будет сформирован по справочнику "Категории номенклатуры".

Таким образом справочник номенклатуры может содержать один набор папок, а иерархия прайс-листа может отличаться. Примером использования могут быть случаи, когда папки номенклатуры заводились по производителям, а прайс-лист необходимо построить по продукции и фактически объединить всех производителей в одну группу прайс-листа.

По кнопке "Запомнить настройки" выполняется сохранение настроек прайс-листа.

Настроенный прайс-лист можно сохранить как "Индивидуальный" или "Общий" для всех пользователей. Переключатель принадлежности прайс-листа  не вводит жестких ограничений для пользователей, служит для быстрого отбора "своих" настроек.

Для каждого пользователя можно настроить поведение программы при переходе со ссылке "Прайс-листы": список сохраненных прайс-листов или непосредственно обработку формирования.

При этом, если количество элементов в справочнике менее двух, опция не учитывается и открывается форма обработки.

 

Цены

Группа содержит настройку "Период: (дата)", на который будут получены цены. Если значение не указано, будут выбраны актуальные данные.

 Если в программе включено использование нескольких валют группа будет дополнена реквизитами "Валюта" и "Дата курса". Цены во всех колонках будут пересчитаны по курсу на указанную дату в текущую валюту.

 

Колонки

Группа содержит настройки колонок, которые будут выведены в прайс-лист:

По ссылке "Колонки" выполняется подбор цен, которые будут выведены в отчет ("Цена поставщика 1", "Цена поставщика 2" и т.д.).

С помощью флажка "Исключить пустые виды цен" можно скрыть колонки (виды цен контрагентов), для которых не записаны цены.

 

Фильтры

В группе "Фильтры" с помощью флажков можно задать условия, при которых в прайс-лист будет выводиться только номенклатура, которая отсутствует на складах, или отображаться номенклатура без цен.

При установке флажка "Сравнить цены контрагентов" в обработке будут отображаться строки по условию сравнения цен – например, только строки, в которых цена "Поставщика А" меньше цены "Поставщика Б".  оптовой.

По ссылке "Еще фильтры" доступна форма "Настройки отбора номенклатуры", позволяющая настроить универсальные отборы по любому реквизиту номенклатуры, в том числе папкам, ценовым группам, категориям.

 

Оформление

В группе собраны настройки флажков, которые указывают, какие ячейки табличного документа будут выделены цветом (промаркированы). Например, пользователь может указать, что необходимо выделить минимальную цену.

Для прайс-листа может быть выбран один тип маркировки. Маркировки "Минимум" и "Максимум" доступны, если выбрано несколько видов цен. 

 

Прочее

Из группы можно перейти к отчету "Исключенная номенклатура" и просмотреть список номенклатуры, которая не используется в прайс-листах. Важно отметить, что сам признак "Не использовать в прайс-листах" хранится в карточке номенклатуры.

По ссылке "О прайс-листе" заполняется дополнительная информация о прайс-листе:

  • дата, которая будет указываться как дата формирования прайс-листа.

  • описание, которое размещается в подвале прайс-листа. Может содержать дополнительную текстовую информацию, например, краткое памятку об условиях закупки товаров.
  • правила сортировки групп и строк номенклатуры.

С помощью переключателя "Открывать как обработку/открывать как список" можно указать, что открывать пользователю по команде "Прайс-листы поставщиков".
