Ассистент управления нашей фирмой

В программе реализован  ассистент управления фирмой – Даша. Ассистент, как и обычный сотрудник, выполняет поручения в программе. 

Например, при обработке заказа Даша может изменить состояние заказа, написать об этом в чате документа и сообщить об изменениях менеджеру.

Навыки Даши:

  • изменять состояние заказа покупателю, поставщику или на производство;
  • оповещать сотрудников, отправляя сообщения во внутреннем чате;
  • оповещать клиентов с помощью писем и SMS;
  • контролировать работу других пользователей по заданным сценариям;
  • рассчитывать налоги по УСН, патенту и страховые взносы ИП. Формировать декларацию по УСН, отчетность по сотрудникам, алкогольную декларацию № 12 (пиво).

 

           


Работа с ассистентом

В  левой части карточки ассистента управления доступны его описание, просмотр истории действий и ссылка, по которой пользователь может предложить новый навык.

В правой части отображаются навыки ассистента. Все навыки разделены на группы, переключаться между которыми можно с помощью кнопок. Активные задачи отмечаются в карточке ассистента зелеными галочками.

Подробное описание задания ассистента можно найти в карточке задания и справке (для перехода нажмите F1 в карточке задания). 

 Пример справки "Изменение состояния заказа при оплате"

Просмотр карточки ассистента и порученных ему задач доступен всем пользователям. Назначать задачи ассистенту может только пользователь с правами "Администратор".
