Помощник настройки кассового места

Помощник настройки кассового места организует работу Администратора по настройке кассового места. Помощник можно запустить под администратором двумя способами:Продажи – Сервис - Помощник настройки кассового места или Администрирование – Продажи - Помощник настройки кассового места. При работе с помощником будут последовательно созданы записи в нужных справочниках, рабочее место для выбранного пользователя, подключено оборудование, настроены виды платежей, принимаемых на данном кассовом месте. Помощник может использоваться также для редактирования уже выполненных настроек.

 

При настройке рабочих мест для вновь открываемого магазина, необходимо до начала работы помощника создать запись в справочнике Магазины. Перед началом работы с помощником необходимо войти в операционную систему под именем пользователя, для которого настраивается кассовое место. Записи в справочнике Рабочие места создаются автоматически для данного компьютера и пользователя операционной системы. Поэтому для того, чтобы сократить объем работ по настройке кассовых мест удобно создать одного пользователя операционной системы, например, Кассиры, если это позволено политикой безопасности компании. Для осуществления же контроля действий кассира можно вести пользователей информационной базы Розницы в разрезе сотрудников магазина.

Программа поможет Администратору создать записи в справочниках в порядке зависимости одного от другого, проверит непротиворечивость данных и поможет исправить допущенные ранее ошибки. Проверит установленные драйверы оборудования, в случае необходимости создаст объекты учета денежных средств, кассы ККМ или операционные кассы магазина для каждой из указанных организаций. Напомнит о необходимости внести эквайринговые терминалы в списки, связанные с договорами эквайринга. Следует учесть, что виды платежей с типом платежная карта можно создавать в разрезе договоров эквайринга и ставок комиссий за проведение платежей. В этом случае программа отчет, который покажет сумму комиссионных платежей. В помощник «Настройки кассового места» добавлена возможность создания и редактирования палитры быстрых товаров.

Общие настройки программы

Настройки программы, влияющие на работу кассовых мест:

 

· Флаг Использовать несколько организаций - Возможность ведения нескольких организаций.

· Флаг Использовать подключаемое оборудование - Использование внешнего оборудования - электронные весы, сканеры штрихкода, эквайринговые терминалы и т.д.

· Флаг Использовать оплату платежными картами - Поддержка эквайринговых операций.

· Флаг Использовать оплату банковскими кредитами - Поддержка зачета оплаты банковскими кредитами.

Эти же настройки могут быть определены в основном интерфейсе программы: Раздел "Администрирование" - "Организации и Финансы", "Подключаемое оборудование" и "Продажи".

НСИ (Нормативно-справочная информация)

 

· Магазин – список выбора, для которого производится настройка кассового места

· Флаг Несколько организаций - Все хозяйственные операции осуществляются в разрезе организаций. Выберите одну или несколько организаций, от имени которых будут оформляться документы на данном кассовом месте.

· Организация – список выбора. Для создания новой организации перейдите в "Список организаций

Эти же настройки могут быть определены в основном интерфейсе программы. Раздел "НСИ (Нормативно-справочная информация)" - "Магазины" и "Организации"

Пользователи ИБ

При создании нового пользователя необходимо выбрать профили:

Кассир - для оформления документов продажи,
Кассир операционной кассы - для оформления операций возврата товаров после закрытия смены.
Пользователь синхронизации данных - для осуществления обмена данных.

Уточните дополнительные права пользователя, выбрав только необходимые разрешения.

 

Кнопка Создать позволяет добавить нового пользователя.

 

А кнопка Доп. права позволяет уточнить права пользователя, выбрав только необходимые.

Эти же настройки могут быть определены в основном интерфейсе программы. Раздел "Администрирование" - "Пользователи и права"

Типы оборудования, которые необходимо подключить или изменить.

· Флаг Считыватель магнитных карт

· Флаг Фискальные регистраторы

Рабочее место создается для текущего пользователя операционной системы на данном компьютере. Рекомендуем задать значащее имя рабочего места.

Поле - Наименование рабочего места.

Для просмотра оборудования, подключенного к данному рабочему месту, перейдите в список Подключаемое оборудование

Эти же настройки могут быть определены в основном интерфейсе программы. Раздел Администрирование - Подключаемое оборудование.

Касса ККМ

 

Касса КММ необходима для разделения учета денежных средств по организациям, магазинам и рабочим местам. Необходимо настроить следующие поля.

· Магазин

· Рабочее место

· Организация

· Тип кассы

· Наименование

· Печать

· Серийный номер

· Регистрационный номер

· Подключаемое оборудование - Печатающее устройство

· Распределение выручки по секциям ФР

· Шаблон чека

· Ширина ленты - Используется для печати товарного чека

· Касса (ККМ)

Эти же настройки могут быть определены в основном интерфейсе программы. Раздел "НСИ (Нормативно-справочная информация)" - "Кассы ККМ"

Касса

Касса необходима для учета движения денежных средств в разрезе хозяйственных операций с помощью документов РКО И ПКО. Необходимо настроить следующие поля.

 

· Магазин

· Организация

· Наименование

· Флаг - Пробивать чеки по кассе ККМ

· Касса (ККМ)

· Шаблон РКО

· Шаблон ПКО

· Касса

Эти же настройки могут быть определены в основном интерфейсе программы. Раздел "НСИ (Нормативно-справочная информация)" - "Кассы"

Виды оплат

Виды оплат по типам Платежная карта и Банковский кредит рекомендуется создавать в разрезе ставок комиссии по договорам эквайринга или предоставления кредита.

 

Эти же настройки могут быть определены в основном интерфейсе программы. Раздел "Продажи" - "Виды оплат".

Эквайринг

Для организации расчетов платежными картами необходимо создать договоры эквайринга. Для Формирования списка эквайринговых терминалов существующего договора, выберите нужный договор.

 

Эквайринговый терминал служит для регистрации Факта установки платежного терминала для осуществления расчетов. Может работать как с подключением оборудования к информационной системе, так и без подключения.

Эти же настройки могут быть определены в основном интерфейсе программы. Раздел "Финансы" - "Договоры эквайринга".

Настройки РМК

Настройка палитры быстрых товаров

 

Рекомендуем выбрать «Настройку РМК» для данного компьютера.

Эти же настройки могут быть определены в основном интерфейсе программы. Раздел "Администрирование" - "Продажи" - "Настройки РМК".
