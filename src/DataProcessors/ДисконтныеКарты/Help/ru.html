Обработка «Дисконтные карты»

Обработка "Дисконтные карты" состоит из двух закладок и объединяет в себе два справочника:

  • "Дисконтные карты". В справочнике  хранится информация о дисконтных картах покупателей. 

  • "Виды дисконтных карт". Справочник хранит настройки для видов дисконтных карт, информацию о предоставляемой по карте скидке, бонусной программе.

Возможность использовать дисконтные карты определяется флагом "Использовать дисконтные карты" в разделе Настройки - Еще больше возможностей - Продажи - Цены и скидки.

В этой справке:

  • Интерфейс
  • Виды дисконтных карт
  • Дисконтные карты

 

 

Интерфейс

На закладке "Дисконтные карты" находится список дисконтных карт, зарегистрированных в информационной базе.

Справа от списка отображается дополнительная информация по выбранной дисконтной карте: процент скидки, владелец, контактная информация.

По гиперссылке "Продажи по дисконтной карте" можно открыть отчет для анализа продаж по выбранной дисконтной карте.

На закладке "Виды дисконтных карт" находится список видов дисконтных карт.

 

Виды дисконтных карт

Рекомендуем начинать работу с дисконтными картами с заполнения информации о видах дисконтных карт. В справочнике определяются правила применения  для дисконтных карт различых видов.

Перейдите на закладку "Виды дисконтных карт" и нажмите кнопку "Создать". Будет открыт диалог для редактирования реквизитов нового вида дисконтных карт.

  Перейти к справке "Виды дисконтных карт"

 

Дисконтные карты

После настройки вида дисконтной карты можно перейти к регистрации самих дисконтных карт.

При создании новой дисконтной карты покупателя устанавливается вид дисконтной карты – элемент справочника "Виды дисконтных карт".  В зависимости от выбранного вида  пользователю будет доступен определенный набор реквизитов.

Реквизит "Владелец" будет доступен, если в виде карты установлен флаг "Именная карта". В качестве владельца дисконтной карты необходимо выбрать контрагента, которому она выдана.

Если тип карты "Штриховая", то программа потребует заполнить штрихкод. Для типа карты "Магнитная" - магнитный код. Для типа карты "Смешанная"  потребуется заполнить оба значения.

Наименование дисконтной карты (поле "Наименование") заполняется автоматически, но может быть изменено вручную.

  Перейти к справке "Дисконтные карты"
