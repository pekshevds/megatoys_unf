Обработка "Прайс-лист организации"

Список товаров, которые реализует компания, с ценами и краткими характеристиками можно сформировать с помощью инструмента "Прайс-листы организации".

Прайс-листы предназначены для просмотра и редактирования отпускных цен компании.

Вы можете сформировать несколько прайс-листов организации с различными ценами на одни и те же товары в зависимости от объемов реализации, сезонности, и других условий.

В этой справке:

  • Внешний вид;
  • Установка цен из прайс-листа;
  • Настройки прайс-листа;
    • Цены;
    • Колонки;
    • Фильтры;
    • Оформление;
    • Прочее.

 Важно! Отображение некоторых реквизитов обработки "Прайс-лист организации" зависит от включенных возможностей программы. Активировать дополнительные возможности можно в разделе Настройки – Еще больше возможностей.

 

 

Внешний вид

Прайс-лист представляет собой таблицу с перечнем товаров и ценами на них.

Цены в прайс-листе заполняются по выбранному в прайс-листе виду цен.

 Перейти к справке "Виды цен"

 

Есть три варианта представления прайс-листа в программе:

  • Прайс-лист вида "Полотно" является классическим вариантом компоновки данных, при котором номенклатура, ее описание, цены и другая информация располагаются в строку, а сам прайс-лист выглядит в форме одной монолитной таблицы.

  • Прайс-лист вида "Две колонки"  представляет собой способ подачи информации с максимальной компактностью. В связи с этим имеет некоторые ограничения, например: выводится только одна колонка с ценой, отсутствует возможность вывода картинки и других второстепенных данных. 

  • В прайс-листе вида "Диафильм" выводятся карточки с изображением номенклатуры, по внешнему виду напоминает каталог товаров. Основное назначение – предоставить пользователю максимально наглядный прайс-лист, где номенклатура представлена в виде картинок.

В шапке можно разместить название компании, контакты, реквизиты. Также можно добавить ключевые преимущества, условия поставки и многое другое.

Прайс-листы можно сохранять в виде файлов на компьютер или отправлять непосредственно из программы с помощью кнопок на панели команд.

Важно! Обработка "Прайс-листы организации" не хранит информацию о товарах, ценах и способах их отображения, это только инструмент для построения прайс-листа. Все настройки хранятся в справочнике "Прайс-листы". 

 Перейти к справке "Номенклатура"

 

Установка цен из прайс-листа

Установить новую цену на товар можно кликнув на ячейку с ценой в прайс-листе.

Откроется форма "Новая цена". Данные в полях "Период", Вид цен, Номенклатура, Характеристика (если необходимо) заполнятся автоматически по данным ячейки прайс-листа. Установите новую цену и нажмите "Записать и закрыть"

После записи цена регистрируется непосредственно в регистре "Цены номенклатуры". Данный регистр содержит информацию о ценах на товары и услуги и историю их изменений.

Назначение цен сразу для списка товаров выполняется в документе "Установка цен".

 Перейти к статье "Установка цен"

 

Настройки прайс-листа

Настройка прайс-листа выполняется с помощью правой панели.

Настройки прайс-листов собраны в группы:

  • Цены,
  • Колонки,
  • Фильтры,
  • Оформление,
  • Прочее.

Поле "Рабочий прайс-лист" позволяет быстро выбрать (переключить) прайс-лист для работы.

Напротив поля располагаются кнопки "Создать" и "Скопировать" с помощью которых можно быстро подготовить новые прайс-листы.

Список всех прайс-листов организации находится по ссылке "Показать все" в выпадающем меню поля "Рабочий прайс-лист".

Для прайс-листа предоставляется возможность выбора вида компоновки: полотно, две колонки, диафильм.

Опция "Иерархия" позволяет выбрать, по какому справочнику построить дерево прайс-листа.

Кроме классического варианта, с иерархией по папкам номенклатуры, присутствует возможность построить прайс-лист по элементам ценовых групп или категорий номенклатуры. Справочник номенклатуры может содержать один набор папок, а иерархия прайс-листа может отличаться.

По кнопке "Запомнить настройки" выполняется сохранение настроек прайс-листа.

Настроенный прайс-лист можно сохранить как "Индивидуальный" или "Общий" для всех пользователей. Переключатель принадлежности прайс-листа  не вводит жестких ограничений для пользователей, служит для быстрого отбора "своих" настроек.

Для каждого пользователя можно настроить поведение программы при переходе со ссылке "Прайс-листы": список сохраненных прайс-листов или непосредственно обработку формирования.

При этом, если количество элементов в справочнике менее двух, опция не учитывается и открывается форма обработки.

 

Цены

Группа содержит настройку "Периода (дата)", на который будут получены цены. Если значение не указано, будут выбраны актуальные данные.

 Если в программе включено использование нескольких валют, группа будет дополнена реквизитами "Валюта" и "Дата курса". Цены во всех колонках будут пересчитаны по курсу на указанную дату в текущую валюту.

 

Колонки

Группа содержит настройки колонок, которые будут выведены в прайс-лист. Для каждого из трех видов прайс-листов состав задаваемых реквизитов различен.

 Настройки разделены на два типа:

  • Состав реквизитов номенклатуры. Какие данные номенклатуры необходимо указать в прайс-листе (наименование, картинка, страна происхождения, новинка и т.д.).
  • Колонки видов цен. Какие виды цен будут выведены в документ (оптовая, розница и т.д.). Если в виде цен нет записей, он не отображается при включенной настройке "Исключить пустые виды цен".

Признак новинки, количественные границы остатков (много/мало) хранятся в карточке номенклатуры, как другие данные о товаре.

 

Фильтры

В группе "Фильтры" с помощью флажков можно задать условия, при которых в прайс-лист будет выводиться только номенклатура, которая имеется в наличии, и/или номенклатура без цен.

При установке флажка "Сравнить цены" в документе будут отображаться строки по условию сравнения цен – например, только строки, в которых розничная цена равна оптовой.

По ссылке "Еще фильтры" доступна форма "Настройки отбора номенклатуры", позволяющая настроить универсальные отборы по любому реквизиту номенклатуры, в том числе папкам, ценовым группам, категориям.

 

Оформление

В группе собраны настройки флажков, которые указывают, какие ячейки табличного документа будут выделены цветом (промаркированы). Например, пользователь может указать, что необходимо выделить минимальную цену.

Для прайс-листа может быть выбран один тип маркировки. Маркировки "Минимум" и "Максимум" доступны, если выбрано несколько видов цен. 

Опция "Условная цена" сравнивает цену со значением, заданным пользователем.

Выделение цветом по условию не используется в прайс-листах с вариантом печати "Диафильм (слайды)".

 

Прочее

Из группы можно перейти к отчету "Исключенная номенклатура" и просмотреть список номенклатуры, которая не используется в прайс-листах. Важно отметить, что сам признак "Не использовать в прайс-листах" хранится в карточке номенклатуры.

По ссылке "О прайс-листе" заполняется дополнительная информация о прайс-листе:

  • дата, которая будет указываться как дата формирования прайс-листа;
  • организация, по данным которой будет заполняться контактная информация;
  • описание, которое размещается в подвале прайс-листа. Может содержать дополнительную текстовую информацию, например краткое описание заключения сделок;
  • контакты автора. Добавить можно только в индивидуальные прайс-листы;
  • правила сортировки групп и строк номенклатуры.
