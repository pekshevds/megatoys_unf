Справочник "Рабочие смены"

Справочник предназначен для хранения информации о рабочих сменах складов и участвует в планировании рабочего времени сотрудников.

Список рабочих смен открывается по ссылке Персонал - Рабочие смены.

На форме элемента рабочей смены указывается Наименование смены, Склад по которому будет выполняться дальнейшее планирование и список должностей. В списке должностей выбираются должности, которые должны быть задействованы в работе на этой смене, так же указывается количество сотрудников с этой должностью необходимых для работы на смене.
