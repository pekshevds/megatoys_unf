Справочник «Виды дисконтных карт»


В программе предусмотрена возможность предоставления различных скидок (наценок) по дисконтным картам. В справочнике определяются правила применения  для дисконтных карт разных видов.

Возможность использовать дисконтные карты определяется флагом "Дисконтные карты" в разделе Настройки - Еще больше возможностей - Продажи - Цены и скидки.

В программе можно применять обезличенные и именные карты. Карты могут быть собственного производства или других производителей, например, социальные картыли карты "дружественных" торговых сетей.

Если установлен флаг "Именная карта", то карта считается именной. При создании карты такого вида обязательно указывается владелец карты (элемент справочника "Контрагенты").

Для вида карты можно указать бонусную программу. По картам такого вида помимо предоставления скидки могут начисляться и списываться бонусные баллы согласно условиям бонусной программы.

Для вида дисконтной карты необходимо выбрать "Тип":

  • Штриховая. Для карт данного вида будет доступно считывание с помощью сканера штрихкода.

  • Магнитная. Для карт данного вида будет доступно считывание с помощью считывателя магнитных карт.

  • Смешанная. Для смешанных карт доступны оба вариант считывания.

По карте лояльности могут быть предоставлены как фиксированные или накопительные скидки. Скидка по дисконтной карте настраивается в справочнике "Виды автоматических скидок, наценок".

Для применения скидки по дисконтной карте, условием предоставления скидки должно быть указано "Предъявлена дисконтная карта".
