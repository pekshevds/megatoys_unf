Справочник "Сотрудники"


Возможности и реквизиты данного справочника отличаются в приложениях "1С:УНФ" и "1С:Розница". 
Если в справке напротив возможности или элемента интерфейса есть иконка, то описанная опция доступна только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Справочник предназначен для хранения информации о сотрудниках организации. Каждому сотруднику соответствует определенное физическое лицо.

В этой справке:

  • Заполнение карточки сотрудника
  • Штрихкод сотрудника
  • Прием и увольнение сотрудника

 

 

Заполнение карточки сотрудника

В рамках учета персонала мы сталкиваемся с двумя классами тесно связанных объектов учета: сотрудниками и физическими лицами. Список физических лиц, как являющихся сотрудниками компании, так и не являющихся, хранится в справочнике "Физические лица".

Каждому сотруднику соответствует определенное физическое лицо. При этом, один и тот же человек (иначе физическое лицо) может рассматриваться как несколько объектов учета (сотрудников).

Например, некое физическое лицо может поработать в организации и уволиться, а затем по прошествии некоторого времени снова быть принятым на работу. В этом случае в программесоздается два сотрудника, принятые в разное время и на разные рабочие места, и они будут связаны с одним физическим лицом.

При создании нового элемента справочника предоставляется выбор:

  • Создать нового сотрудника и внести его личные данные в справочник физических лиц. В этом случае кроме сотрудника будет создан элемент справочника "Физические лица" и установлена связь между ними.
  • Создать нового сотрудника, выбрав его из справочника физических лиц. При выборе заполняется наименование сотрудника и устанавливается связь с элементом справочника "Физические лица".

В карточке сотрудника указываются сведения, связанные с физическим лицом сотрудника: пол, дата рождения,паспортные данные, адрес по прописке, гражданство. Эти данные – сквозные для справочников "Сотрудники" и "Физические лица".

Также в справочнике содержатся данные о сотруднике как налогоплательщике (ИНН), страховой номер сотрудника в пенсионном фонде (СНИЛС) .

Группа "Работа" отражает информацию о типе занятости сотрудника: основное место работы или совместительство, и данные о приеме сотрудника на работу: дата, должность и др. Данные заполняются на основании информации документов "Прием на работу" и кадровое перемещение.

Значение поля "Должность" является элементом справочника "Должности". В этом справочнике хранится список названий должностей, которые могут занимать сотрудники компании.

С помощью ссылок "Принять на работу"/"Уволить" из карточки сотрудника можно оформить соответствующие кадровые документы.

В группе "Основной банковский счет" можно указать банк и номер основного счета сотрудника для перечисления заработной платы.

Группа "Вычеты" дает возможность заполнить вычеты по НДФЛ на каждого сотрудника: вычеты на детей, имущественные, социальные и личные вычеты.

 

По гиперссылке  Счета учета  на панели команд заполняются счета учета для расчетов с сотрудником.

Реквизиты: 

  • С персоналом. Указывается счет расчетов с персоналом. Выбирается из Плана счетов "Управленческий".
  • С подотчетником. Указывается счет расчетов с подотчетником. Выбирается из Плана счетов "Управленческий".
  • По перерасходу. Указывается счет расчетов по перерасходу. Выбирается из Плана счетов "Управленческий".

По кнопке "Восстановить по умолчанию" устанавливаются дефолтные значения.

 

Штрихкод сотрудника

Настройки штрихкодов используются для поиска сотрудника по штрихкоду внутри системы. Например, при авторизации сотрудника в  рабочем месте кассира (РМК).

Для добавления информации по штрихкоду и номеру карты можно использовать "Сканер штрикодов" и "Считыватель магнитных карт". Оборудование должно быть подключено через Подключаемое оборудование.

 

Прием и увольнение сотрудника

Для того чтобы зарегистрировать физическое лицо в качестве сотрудника организации, оформляется документ "Прием на работу".

Если при создании нового сотрудника в группе "Работа" нажать на ссылку "Принять на работу" и заполнить все поля карточки, подчеркнутые красным, автоматически сформируется "Прием на работу".

Для того чтобы зарегистрировать увольнение сотрудника организации и прекратить действие плановых начислений оформляется документ "Увольнение". Создать документ увольнения  можно из карточки сотрудника (группа "Работа") по гиперссылке "Уволить".

Все документы приема и увольнения, а также кадровые перемещения данного сотрудника можно просмотреть, воспользовавшись гиперссылкой "Сведения о сотрудниках" на панели навигации карточки сотрудника.
