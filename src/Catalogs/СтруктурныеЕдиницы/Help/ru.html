Справочник "Структурные единицы"


Возможности и реквизиты данного справочника отличаются в приложениях "1С:УНФ" и "1С:Розница". 
Если в справке напротив поля или элемента интерфейса есть иконка, то элемент доступен только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Справочник "Структурные единицы" предназначен для хранения информации об организационно-структурных единицах компании:

  • административно-управленческих и производственных подразделениях;
  • складах и магазинах.

В справочнике "Структурные единицы" предусмотрен переключатель, позволяющий просматривать отдельно "Склады и магазины", "Подразделения" или "Всю структуру".


Важно! Для структурных единиц устанавливается привязка к конкретной организации если включена синхронизация с "1С:Бухгалтерией". Организация требуется только для синхронизации не используется в учете.

 

Подразделения

Если необходимо вести учет по нескольким подразделениям, в настройках раздела "Компания" нужно включить опцию "Несколько подразделений". При включенной опции появляется возможность указывать подразделения в документах.

Для каждой организационно-структурной единицы указываются ее наименование и тип "Подразделение".

В группе "Печать" можно указать параметры для формирования печатных форм (счета-фактуры, универсального передаточного документа и других).

Реквизит "Головная организация" заполняется только для подразделений. Если головная организация одновременно заполнена для Организации и Подразделения, в документ будет подставлены данные из карточки организации.

В группе "Адреса, телефоны" можно указать фактический адрес подразделения и телефон.

Из формы организационно-структурной единицы по гиперссылке "Просмотр и редактирование параметров автоперемещения запасов" вызывается дополнительный диалог ввода и изменения параметров перемещения запасов для подстановки в документы учета запасов по умолчанию.


Склады и магазины

Список "Склады и магазины" содержит сведения об организационно-структурных единицах компании типов:

  • "Склад",
  • "Розничный склад",
  • "Розничный склад (суммовой учет)",
  • "Магазин или группа складов".

В карточке организационно-структурной единицы указываются тип, розничный вид цен, материально ответственное лицо, данные для подстановки в печатные формы, адреса и телефоны подразделения.

Из формы организационно-структурной единицы по гиперссылке "Просмотр и редактирование параметров автоперемещения запасов" вызывается дополнительный диалог ввода и изменения параметров перемещения запасов для подстановки в документы учета запасов по умолчанию.


 Ячейки склада

Учет запасов на складе может осуществляться в разрезе структуры мест хранения (стеллажей, участков и т. д.), объединенных под общим названием "ячейки склада".

Чтобы включить возможность ведения складского учета в разрезе ячеек, необходимо установить флажок Ячеистые склады в форме параметров учета Склад в разделе Настройки.

В прикладном решении допускается любое количество ячеек.

Каждой структурной единице (складу, магазину, подразделению) можно назначить произвольное количество ячеек склада, перейдя к форме подчиненного справочника "Ячейки склада". Для каждой ячейки задается наименование и разрешается неограниченное количество уровней вложенности.

Ордерный учет

Учет запасов может осуществляться в разрезе складских ордеров. Для этого необходимо в "Настройки->Еще больше возможностей -> Склад" включить опцию "Ордерные склады". После включения опции в форме элемента структурной единицы появится группа "Ордерный учет".

В данной группе можно указать будет ли использоваться ордерный учет для текущей структурной единицы. Так же в группе указывается при каких операциях (поступление, перемещение, отгрузка) будет необходимо вводить складские ордера и учитывать ли остатки по документам при их введении.

Доступны следующие настройки учета остатков по документам:


  • "Не используется" - остатки к приходу/расходу по документам со складов не фиксируются. 
  • "По накладным" - При поступлении/отгрузке на основании первичных документов необходимо оформить приходные/расходные складские ордера. Контролируется остаток к оформлению в разрезе первичных документов.
  • "По складским ордерам" - При поступлении/отгрузке товаров первичными документами являются приходные/расходные складские ордера. На их основании вводятся дальнейшие документы поступления/отгрузки. Контролируется остаток к оформлению в разрезе первичных документов.
