Справочник "Контакты"

Справочник предназначен для хранения физических лиц, с которыми ведется работа в компании. Контакт может представлять одного или нескольких контрагентов.


   

В карточке контактного лица указываются представление контактного лица, контактные данные, связанный контрагент и детали, такие как пол, дата рождения, источник привлечения и прочее. В карточке контакта можно сохранить файлы, возникшие при взаимодействии.

В карточке контактного лица указывается информация, характеризующая контакт:

  • Имя Фамилия. Представление контакта для обращения к контактному лицу при взаимодействии. Предусмотрена возможность уточнить правила склонения ФИО, например, для подстановки в рассылках. 
  • Юр. данные. Дата рождения, пол и данные документа, удостоверяющего личность.

В группе Адреса, телефоны заполняется контактная информация для связи с контактом. Для каждого контакта можно указать дополнительную информацию – Skype, социальные сети, мессенджеры.

В группе Контрагенты устанавливается связь контакта с контрагентом и отображается информация о контрагентах, с которыми контакт связан на данный момент. Контакты могут быть привязаны к одному или нескольким элементам справочника "Контрагенты". При этом, если информации о контрагентах, с которыми связан контакт нет, можно не привязывать контакт ни к одному из контрагентов. 

Для каждого контрагента, с которым связан контакт может быть указана отдельная должность.

Чтобы узнать всю историю связей контакта нажмите на кнопку Показать в списке возле поля Контрагент. Откроется список контрагентов с дополнительным блоком История связей в правой панели.

В поле В группе можно определить, в какую группу будет входить текущий контакт. При ручном изменении реквизита "Группа" происходит перемещение текущего элемента из старой группы в новую.

В группе Детали указывается дополнительная информация о контакте: роли контактного лица в компании, источник его привлечения, любая дополнительная информация, а также сотрудник, создавший контакт.

Если контакт недействительный, его можно скрыть из списков, установив флажок Недействителен. Информация о нем сохраняется в программе.

Контактную информацию можно распечатать по кнопке Печать из карточки или списка справочника "Контрагенты". Контактная информация выводится либо в виде строки по каждому контрагенту, либо в виде отдельной области, в зависимости от варианта печатной формы: список или карточка.
