Справочник "Способы оплаты"

Справочник предназначен для хранения перечня способов оплат собственных организаций.

В зависимости от настроек программы, способ оплаты может иметь один из следующих типов:

  • Эквайринг - используется в качестве эквайрингового терминала организации. Создается вручную.
  • ЮKassa - используется для интеграции с платежной системой "ЮKassa". Создается автоматически при настройке интеграции.
  • СБП - используется для интеграции с Системой Быстрых Платежей. Создается автоматически при настройке интеграции.
  • Другая платежная система - используется для прочих платежных систем, например при интеграции с интернет-магазинами.

Для типа Эквайринг доступна табличная часть Виды платежных карт. В ней указывается перечень карт или платежных систем, которые поддерживает эквайринговый терминал. Кроме того, указываются проценты комиссии:

  • Продажа - комиссия, которую берет банк за проведение операции оплаты.
  • Отмена - комиссия, которую берет или возвращает банк за возврат оплаты в течение той же кассовой смены, в которой была и сама продажа.
  • Возврат - комиссия, которую берет или возвращает банк возврат оплаты по продаже, которая была в другой кассовой смене.

Если банк возвращает комиссию за возврат или отмену платежа, ее необходимо указать со знаком "минус".
