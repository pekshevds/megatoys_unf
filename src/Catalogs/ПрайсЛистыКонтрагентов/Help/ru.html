Справочник "Прайс-лист контрагентов (поставщиков)"

Справочник предназначена для хранения настроек (правил) формирования прайс-листов с входящими ценами.

В справочнике хранятся настройки, которые использует обработка "Прайс-листы контрагентов" при построении прайс-листа поставщиков.

 Перейти к справке обработки "Прайс-листы контрагентов"

 

 

Настройки прайс-листа

Для прайс-листа обязательно заполняется наименование. С помощью уникального имени пользователь может быстро выбрать рабочий прайс-лист в обработке "Прайс-листы контрагентов". 

Настройки прайс-листов собраны в группы:

  • Цены,
  • Колонки,
  • Фильтры, отборы,
  • Оформление,
  • Дополнительные сведения о документе.


По ссылке "Иерархия"можно выбрать, по какому справочнику формируется дерево прайс-листа:

  • Иерархия групп номенклатуры. Отчет будет сформирован по папкам номенклатуры.

  • Иерархия ценовых групп. Отчет будет сформирован по справочнику "Ценовые группы".

  • Иерархия категорий номенклатуры. Отчет будет сформирован по справочнику "Категории номенклатуры".

Таким образом справочник номенклатуры может содержать один набор папок, а иерархия прайс-листа может отличаться. 

Примером использования могут быть случаи, когда папки номенклатуры заводились по производителям, а прайс-лист необходимо построить по продукции и фактически объединить всех производителей в одну группу прайс-листа.

 

Цены

Группа настроек содержит поле "Цены соответствуют периоду:". В данном поле устанавливается дата, на которую будут получены цены. Если значение не указано, будут выбраны актуальные данные.

Если в программе включено использование нескольких валют группа будет дополнена реквизитами "Валюта" и "курс на дату".

Цены во всех колонках будут пересчитаны по курсу на указанную дату в текущую валюту.

 

Колонки

Группа содержит настройки колонок, которые будут выведены в прайс-лист.

В таблице выполняется подбор цен, которые будут выведены в обработку ("Цена поставщика 1", "Цена поставщика 2" и т.д.).

С помощью флажка "Исключить пустые виды цен" можно скрыть колонки (виды цен контрагентов), для которых не записаны цены.

 

Фильтры, отборы

В группе "Фильтры" с помощью флажков можно задать условия, при которых в прайс-лист будет выводиться только номенклатура, которая отсутствует на складах, или отображаться номенклатура без цен.

При установке флажка "Сравнить цены контрагентов" в обработке будут отображаться строки по условию сравнения цен – например, только строки, в которых цена "Поставщика А" меньше цены "Поставщика Б". оптовой.

В таблице задаются настройки отбора номенклатуры, позволяющие настроить универсальные отборы по любому реквизиту номенклатуры, в том числе папкам, ценовым группам, категориям.

 

Оформление

В группе собраны настройки флажков, которые указывают, какие ячейки табличного документа будут выделены цветом (промаркированы). Например, пользователь может указать, что необходимо выделить минимальную цену.

Для прайс-листа может быть выбран один тип маркировки. Маркировки "Минимум" и "Максимум" доступны, если выбрано несколько видов цен. 

Опция "Условная цена" сравнивает цену со значением, заданным пользователем.

Выделение цветом по условию не используется в прайс-листах с вариантом печати "Диафильм (слайды)".

 

Дополнительные сведения о документе

По ссылке "Дополнительные сведения о документе" заполняется дополнительная информация о прайс-листе:

  • дата, которая будет указываться как дата формирования прайс-листа.
  • правила сортировки групп и строк номенклатуры.
  • описание, которое размещается в подвале прайс-листа. Может содержать дополнительную текстовую информацию, например, краткое памятку об условиях закупки товаров.
