Справочник "Характеристики номенклатуры"

Справочник содержит информацию о дополнительных характеристики элементов номенклатуры: цвет, размер и т.п., в разрезе которых можно вести учет запасов.

Например, для описания характеристик товаров категории "Обувь" могут быть использованы такие виды свойств, как размер, цвет, полнота. Для описания товаров категории "Холодильники" используются такие виды свойств, как габариты, объем морозильной камеры, цвет отделки.

Использование характеристик позволяет хранить товары, например, разного цвета в одной карточке номенклатуры и, таким образом, минимизировать объем справочника "Номенклатура".

В этой справке:

  • Ввод характеристик
  • Особенности учета по характеристикам
  • Копирование характеристик

   

Ввод характеристик

Для использования характеристик в программе должен быть установлен флажок Характеристики номенклатуры в разделе Настройки – Еще больше возможностей – Общее.

Характеристики назначаются для позиций номенклатуры, для которых установлен флажок Вести учет по характеристикам.

Информация о характеристиках и список их свойств хранится в программе в привязке к категориям номенклатуры.  Чтобы для товара можно было задавать характеристики, необходимо предварительно настроить категорию номенклатуры,  к которой относится товар. Виды свойств и свойства товара, из которых формируется характеристика , хранятся в карточке категории.

Характеристики могут быть индивидуальными для товара или едиными для категории. Общие характеристики категории отображаются в списке характеристик номенклатуры с флажком Из категории. С помощью фильтра, расположенного справа от таблицы, можно выбрать, какие характеристики показывать в списке.


 Перейти к справке "Категории номенклатуры"


При создании новой характеристики заполняются:

  • Наименование и Наименование для печати. Заполняются автоматически по шаблону, но можно отредактировать вручную. Есть возможность задать пользовательские шаблоны наименований.
  • Артикул. Используется для поиска и подбора номенклатуры в документах  и списках, а также для сопоставления характеристик при загрузке из Excel.
  • Значения свойств для товара (например: черный, 36). Виды свойств подставляются из категории номенклатуры (например: цвет, размер).

Для характеристики можно добавить изображение. Изображения характеристик отображаются в прайс-листах, выгружаются на сайты и веб-витрины (>Mag1c, Магазин ВКонтакте).

Одну из характеристик можно указать как основную. Она будет автоматически подставляться в документы при выборе товара. Такая возможность упростит ввод документов при использовании обязательного заполнения характеристик.
Чтобы сделать характеристику основной, нужно выбрать ее в списке и нажать кнопку Использовать как основную. Слева от характеристики появится зеленый флажок .

В программе предусмотрена возможность проверки заполнения характеристик в документах. Для этого в карточке товара устанавливается флажок Обязательное заполнение характеристики. Если флажок установлен, при выборе этого товара в документе поле Характеристика будет отмечено обязательным для заполнения (красная пунктирная линия).

Новые характеристики для позиции номенклатуры можно добавлять не только из карточки, но и в момент оформления документа, например, при вводе поступления запасов на склад.

Особенности учета по характеристикам

Характеристики номенклатуры являются одним из разрезов аналитики учета. Они дают возможность анализировать продажи и остатки на складах в разрезе характеристик.

При использовании характеристик в документах отображается дополнительная колонка "Характеристика".

 В списке номенклатуры, а также в форме подбора остатки запасов можно просматривать в разрезе различных характеристик.

При проведении документов, отражающих движения товаров (например, "Расходная накладная", "Приходная накладная", "Заказ-наряд" и др.) остатки товаров на складе изменяются для конкретной характеристики.

Копирование характеристик

Быстро ввести характеристики в карточку товара позволяет возможность их копирования между товарами с помощью кнопок Скопировать от и Скопировать другим.

Копирование выполняется в специальном окне "Копирование характеристик". Правая и левая части формы отличаются в зависимости от выбранной команды:

  • Кнопка "Скопировать от". В левой части окна выберите номенклатуру (или несколько) из которой необходимо скопировать характеристики. В правой отобразятся все характеристики, которые можно скопировать. Отметьте флажками копируемые данные и нажмите кнопку "Скопировать и закрыть".
  • Кнопка "Скопировать другим". В левой части отобразятся характеристики открытой номенклатуры, которые можно скопировать в другую номенклатуру. В список не попадут характеристики категории. Выберите в правой части окна товары, для которых необходимо скопировать характеристики, и нажмите кнопку "Скопировать".
