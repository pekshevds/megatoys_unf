Валюта документа

Дополнительный диалог "Валюта" для заполнения и изменения данных о валюте и курсе валюты докуента открывается при нажатии кнопки "Редактировать валюту" в форме документа.

 

Элементы диалога:

  • Валюта документа. Группа реквизитов:

    • Валюта - валюта документа. Выбирается из справочника "Валюты". Изменение валюты обуславливает автоматический пересчет цен в документе только в том случае, если установлен флажок "Пересчитать цены по курсу". Курс, по которому будут пересчитаны цены, можно установить вручную, в реквизитах Курс и Кратность, которые появляются на форме при установке флажка "Пересчитать цены по курсу".

    • Курс - курс выбранной в документе валюты. Значение реквизита изменяется только при изменении валюты документа и доступно только для просмотра.

    • Кратность - кратность выбранной в документе валюты. Значение реквизита изменяется только при изменении валюты документа и доступно только для просмотра.

  • Налоги. Группа реквизитов:

    • Налогообложение. Тип налогообложения продажи товаров и услуг налогом на добавленную стоимость.

    • НДС. Возможны два варианта заполнения суммы НДС в документах. Если выбран "НДС в сумме", то предполагается, что НДС уже включен в сумму документа. В графе "Всего" отображается сумма с включенным НДС. НДС вычисляется исходя из суммы по каждой позиции и ставки НДС, указанной в документе. Если выбран "НДС сверху", то в сумму по документу (графа "Сумма") не включается НДС.

    • НДС включать в стоимость. 

      Если флажок установлен: 

      При приобретении сумма НДС включается в себестоимость поступивших ценностей. При продаже сумма НДС не выделяется как налог к уплате.

      Если этот флаг не установлен: 

      При приобретении ценностей сумма НДС не включается в их себестоимость, а будет отдельно отражаться на счетах учета НДС для возможности последующего вычета. При реализации будут формироваться дополнительные движения для отражения НДС в балансе, доходах и расходах, а также в расчетах по налогам.
