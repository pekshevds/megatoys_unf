Отчет "Склад"

В отчете отображается актуальное состояние склада и данные для кладовщика.

Отчет позволяет получить полную информацию о поступлении, отгрузке и текущем количестве товаров в выбранном месте хранения.

Отчет может быть сформирован в вариантах:

  • Остатки по складу;
  • Движение товаров по складам;
  • Расхождения по ордерным складам (если включена опция "Ордерные склады").


Для удобного и наглядного анализа данных в программе предусмотрена возможность самостоятельной настройки отчетов и их детализации.

         


Варианты отчета

Остатки по складу

В режиме "Остатки" в отчете не отображается информация по приходу и расходу и выводятся только конечные остатки товаров по выбранному месту хранения.

В отчете можно настроить фильтр для отображения остатков по конкретному складу.

 

Движение товаров по складам

Отчет предназначен для количественного анализа остатков и движений товара за заданный пользователем период. 

В отчете отображается информация о начальных и конечных остатках на складах организации. Также отчет позволяет получить полную информацию о поступлении и отгрузке товаров за выбранный промежуток времени.

Данные представлены в количественном выражении и группируются по складам.

При использовании ордерной схемы по соответствующим складам можно посмотреть информацию о количестве запасов, подлежащих списанию расходными ордерами, и количестве запасов, подлежащих оприходованию приходными накладными.

 

Расхождения по ордерным складам

Когда включена опция "Ордерные склады" (учет запасов разделен на складской и финансовый), отчет можно сформировать в режиме "Расхождения по ордерным склада".

Отчет "Расхождения по ордерным складам" отвечает на вопрос "Какие документы с каким количеством номенклатуры нужно ввести, чтобы в складском и финансовом учете не было расхождений?".

С помощью отчета вы можете быстро найти расхождения между финансовыми и складскими документами, оформленными за выбранный период. Отчет показывает, по каким документам ордерной схемы учета возникли расхождения, с расшифровкой по номенклатурным позициям.

 

Остатки по ячейкам

Когда включена опция "Учет по ячейкам", отчет можно сформировать в режиме "Остатки по ячейкам". Контекстный вариант отчета, открывается из документов заказ покупателя, расходная накладная и справочника номенклатура.

При открытии из документов данные фильтруются по номенклатуре из табличных частей. В отчете видны остатки отфильтрованных запасов в разрезе ячеек.
