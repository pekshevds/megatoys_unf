Отчет "Универсальный отчет"

Универсальный отчет позволяет получить отчет по различным типам данных программы, в том числе по данным, которые в стандартных отчетах недоступны.

Позволяет отображать данные справочников, документов, регистров накопления, регистров сведений, регистров бухгалтерии и их виртуальных таблиц.

Отчет включен в список отчетов раздела "Настройки".

Отчет используется, если необходимо получить данные, но настроить специальный отчет для их получения по каким-либо причинам невозможно.

 

Выбор данных для получения отчета

Основные настройки отчета выведены в шапку.

Чтобы сформировать универсальный отчет:

  • Выберите тип объекта из списка, например, "Регистр накопления". 
  • В зависимости от типа данных, возможно, придется указать дополнительные сведения для получения отчета. Например, для регистра накопления нужно выбрать "Период". 
  • Выберите имя объекта. Например, вид регистра накопления - "Запасы на складах". 
  • Укажите, какие данные показать в отчете. Например, для регистра накопления "Запасы на складах" можно выбрать  имя таблицы: "Основные данные", "Остатки и обороты", "Остатки" или "Обороты".

 

Настройки отчета

Предусмотрена возможность самостоятельной настройки отчета и его детализации. Поменять настройки отчета можно с помощью кнопки "Настройки".

    

 

Настройку отчета можно сохранить для дальнейшего использования.

Для этого по кнопке  выберите команду "Сохранить как". В открывшейся форме укажите наименование отчета и вариант доступа к нему (только автору или указанным пользователям).

В дальнейшем для формирования отчета с сохраненными настройками выбирайте его по кнопке "Выбрать другой вариант отчета или сохранить новый".

 

Схема компоновки отчета

Настройки универсального отчета можно перенести из другой информационной базы с помощью загрузки схемы компоновки из файла.

С помощью команд меню "Еще" - "Варианты отчета" можно:

  • Загрузить схему – позволяет загрузить схему компоновки универсального отчета из XML файла. Это расширяет возможности универсального отчета, можно использовать опыт других разработчиков. Выберите путь к файлу со схемой компоновки.
  • Редактировать схему – позволяет вносить произвольные изменения в схему компоновки универсального отчета. Команда работает только в толстом клиенте.
  • Стандартная схема – возвращает настройки универсального отчета по умолчанию. Воспользуйтесь командой, если новая схема компоновки по каким-либо причинам не устраивает.
