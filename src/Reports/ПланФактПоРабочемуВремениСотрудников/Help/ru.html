Отчет "План факт по рабочему времени"


Отчет (Персонал – Отчеты – План/Факт по рабочему времени сотрудников) План-факт по рабочему времени сотрудников для анализа фактически проработанного времени в сравнении с планом. Для каждого сотрудника можно сформировать отчет-расшифровку, уточняющий отклонения фактически проработанного времени от планового.
