Отчет "Движение товаров"

Отчет представляет собой товарную ведомость с детализацией по местам хранения и хозяйственным операциям.

В отчете выводится информация об остатках и оборотах товаров по всем складам, с группировкой по отдельным местам хранения.

Данные представлены в количественном и суммовом выражении.

Данный отчет уникален тем, что отображает движения товаров в разрезе отдельных хозяйственных операций.

С его помощью пользователь может отобрать только движения определенного вида. Например, просмотреть только возвраты, или, наоборот, исключить из отчета внутренние перемещения.
