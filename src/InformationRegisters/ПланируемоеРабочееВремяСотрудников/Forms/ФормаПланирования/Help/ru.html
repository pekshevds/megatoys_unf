Форма "Планирование времени сотрудников"

Форма предназначена для планирования ежедневной работы сотрудников. Форма может быть открыта по ссылке Персонал - Планирование времени сотрудников. Результат сохранения формы будет использоваться для заполнения документа Ежедневный отчет.

В шапке формы можно заполнить следующие реквизиты:

  • Начало недели, дата начала недели, за которую выполняется планирование. Планирование возможно только на неделю.
  • Склад. Фильтр склада, для которого выполняется планирование. Внимание, изменение фильтров приведет к обновлению табличной части, несохраненные данные могут быть потеряны.

  • Организация. Фильтр организации для подбора оформленных в нее сотрудников. Внимание, изменение фильтров приведет к обновлению табличной части, несохраненные данные могут быть потеряны.

  • Подразделение. Фильтр подразделения для подбора оформленных в этой подразделение сотрудников. Внимание, изменение фильтров приведет к обновлению табличной части, несохраненные данные могут быть потеряны.

Команды шапки формы:
  • Прошлая неделя - выполняет переключение на прошлую по календарю неделю. Обновляет табличную часть.
  • Следующая неделя - выполняет переключение на следующую по календарю неделю. Обновляет табличную часть.
  • Записать - записывает данные формы в регистр "Планируемое рабочее время сотрудников"
  • Печать - Запланированное время сотрудников - отображает печатную форму запланированного времени сотрудников. В печать попадают только непустые строки.

Табличная часть формы представляет собой дерево:
  • Первый уровень дерева - склад. На этом уровне указан склад и его график.
  • Второй уровень дерева - рабочая смена.
  • Третий уровень дерева - должность на этом уровне в колонках дней недели указывается количество выбранных сотрудников и общее количество сотрудников для этой должности, указанное в табличной части справочника рабочие смены.
  • Четвертый уровень дерева - сотрудник. Сотрудники подбираются по должности, которая указана в справочнике рабочие смены. На новую неделю подбираются только оформленные сотрудники. В колонках дней недели можно указать рабочее время. По умолчанию заполняется рабочее время из графиков по следующему порядку: Если указан график сотрудника, то рабочие часы подбираются из него (в графике шаблон заполнения), если график сотрудника не указан, то используются часы графика подразделения. В данных колонки так же выводится итоговая сумма часов сотрудника за день по разным должностям и сменам. Например: можно указать сотруднику по должности Продавец 2 часа и по должности Консультант еще 2 часа, итог будет 4 часа за день что будет отражено в колонке дня. Для заполнения строки на неделю или ячейки на один день можно воспользоваться командой "Заполнить выбранную строку на неделю" или "Заполнить выбранный день сотрудника".

Команды табличной части:
  • Заполнить - заполнение выполняется по приоритету графика сотрудника.
    • Весь план - заполняет весь план по всем сотрудникам.
    • Выбранную строку на неделю - заполнить неделю по выбранному сотруднику.
    • Выбранный день сотрудника - заполнить выбранный день по выбранному сотруднику.
  • Добавить сотрудника - добавление сотрудника в табличную часть. Добавлять можно нетрудоустроенных сотрудников, для них данные заполнения будут использованы из графика склада.
  • Перечитать текущий план - обновляет табличную часть текущими данными планирования. Несохраненные данные могут быть потеряны.
